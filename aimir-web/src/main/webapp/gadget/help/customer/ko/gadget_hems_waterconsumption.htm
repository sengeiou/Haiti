AIMIR v4.0 Manual

수도 사용량 조회는 각 가정의 수도 사용량을 기간별로 조회할 수 있는 기능을 제공한다.

 

 

미니 가젯

미니 가젯은 "요약", "기간별 조회" 2개의 탭으로 구성되 어있다.

•요약 : 당일 / 당월의 수도 사용정보를 요약하여 표시한다.
•기간별 조회 : 기간별로 수도 사용정보를 조회할 수 있다.

 

[요약] 탭

수도 사용량의 요약 정보를 표시한다.

•yyyy. m. d 수도 사용정보

    - 일자 : 검침일 기준으로 가장 최근의 일자를 표시한다.

    - 사용요금 : 일자에 해당하는 수도요금을 표시한다.

    - 인센티브 : 일자에 취득한 인센티브를 표시한다.

    - 탄소배출량 : 수도 사용량에 대한 탄소 배출량을 표시한다.

•월 사용요금 비교

    - 송당리 평균 : 이번달 동일 지역 평균 수도요금을 표시한다.

    - 전월 : 전달의 수도요금을 표시한다.

    - 현월 : 현재 달의 수도요금을 표시한다.

    - 타겟 : 수도 에너지 절감 목표에서 설정한 절감 목표 금액을 표시한다.

                (절감 목표 미 설정또는 절감 목표가 지역평균, 전월, 현월 보다 클경우, 그래프에서 표시 되지 않는다.)

 

 [기간별 조회] 탭

주기별(시간/ 일/ 월)로 수도 사용 정보를 조회할 수 있다.

또한 기준일의 동일 지역 시간평균을 타겟으로 표시하여 교차 비교 가능하도록 한다.

그래프에 마우스 오버시 해당 그래프의 상세 설명이 표시 된다.

그래프하단의 일자를 클릭하면, 해당일자의 그래프가 사라진다. 재 클릭시 다시 표시된다.
