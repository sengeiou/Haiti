AIMIR v4.0 Manual

가스 사용량 조회는 각 가정의 가스 사용량을 기간별로 조회할 수 있는 기능을 제공합니다.

 

 

미니 가젯

미니 가젯은 "요약", "기간별 조회" 2개의 탭으로 구성되어있습니다.

•요약 : 당일 / 당월의 가스 사용정보를 요약하여 표시.
•기간별 조회 : 기간별로 가스 사용정보 조회.

 

[요약] 탭

가스 사용정보를 요약하여 표시합니다.

•yyyy. m. d 가스 사용정보

    - 일자 : 검침일 기준으로 가장 최근일자 표시.

    - 사용요금 : 해당 일자에 해당하는 가스요금 표시.

    - 인센티브 : 해당 일자에 취득한 인센티브 표시.

    - 탄소배출량 : 가스 사용량에 대한 탄소 배출량 표시.

•월 사용요금 비교

    - 송당리 평균 : 이번달 동일 지역 평균 가스요금 표시.

    - 전월 : 전달의 가스요금 표시.

    - 현월 : 현재 달의 가스요금 표시.

    - 타겟 : 가스 에너지 절감 목표에서 설정한 절감 목표금액 표시.

                (절감 목표 미 설정또는 절감 목표가 지역평균, 전월, 현월 보다 클경우, 그래프에서 표시 되지 않는다.)

 

[기간별 조회] 탭

주기별(시간/ 일/ 월)로 가스 사용 정보를 조회할 수 있습니다.

 

주기

항목

설명

동작 개요

시간별

주기 콤보박스, 일자 출력

선택한 일(이하, 기준일)에 해당하는 가스 사용정보를 시간별로 조회 한다.

달력 아이콘 또는 이전,다음 화살표 아이콘을 클릭하여 조회하고자 하는 일자를 선택한다.

일자출력 텍스트는 읽기 전용이며 편집 불가능 하다.

검색 버튼

기준일에 해당하는 가스 사용정보를 조회 한다.

버튼 클릭

그래프

기준일과 전일의 가스 사용정보를 시간별로 표시한다.

또한 기준일의 동일 지역 시간평균을 타겟으로 표시하여 교차 비교 가능하도록 한다.

그래프에 마우스 오버시 해당 그래프의 상세 설명이 표시 된다.

그래프하단의 일자를 클릭하면, 해당일자의 그래프가 사라진다. 재 클릭시 다시 표시된다.

텍스트 박스

기준일과 전일의 가스요금, 사용량, 인센티브, 탄소 배출량을 표시한다.
