▪ 요약 : 당일 / 당월의 전기 사용정보를 요약하여 표시.

1) [요약] 탭

전기 사용정보를 요약하여 표시합니다.
 

•yyyy. m. d 전기 사용정보

    - 일자 : 검침일 기준으로 가장 최근 일자 표시.

    - 사용요금 : 해당 일자에 해당하는 전기요금 표시.

    - 인센티브 : 해당 일자에 취득한 인센티브 표시.

    - 탄소배출량 : 전기 사용량에 대한 탄소 배출량 표시.

 

•월 사용요금 비교

    - 송당리 평균 : 이번달 동일 지역 평균 전기요금 표시.

    - 전월 : 전달의 전기요금 표시.

    - 현월 : 현재 달의 전기요금 표시.

    - 타겟 : 전기 에너지 절감 목표에서 설정한 절감 목표 금액 표시.

                (단, 절감 목표 미 설정또는 절감 목표가 지역평균, 전월, 현월 보다 클경우, 그래프에서 표시 되지 않는다.)
