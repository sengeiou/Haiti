AIMIR v4.0 Manual

각 가정의 가스 에너지 절감 목표 설정및 실적 관리 기능을 제공합니다.

가스 에너지 절감 목표는 월단위로 설정가능하며, 절감 실적은 통보기능을 통해서 고객이 설정한 주기로 통보됩니다.

목표 관리

에너지 절감 대상 년/월

오늘 기준으로 1년간의 에너지 절감 목표를 설정할 수 있습니다.

가스 절감 목표를 설정하고자 하는 년과 월을 선택하고, 절감 금액을 입력합니다.

가스 절감 목표 금액

가스 절감 목표 금액을 입력합니다.

설정버튼

고객이 입력한 정보로 에너지 절감 목표를 관리 합니다.

그래프 [목표]

고객이 설정한 목표 금액을 그래프로 표시한다.

그래프 [최고]

고객의 과거 최고 월 가스요금을 그래프로 표시한다.

그래프 [전월]

선택한 절감 대상 년/월 기준으로 전월의 가스요금을 그래프로 표시한다.

그래프 [전년도 동월]

선택한 절감 대상 년/월 기준으로 전년도 동월의 가스요금을 그래프로 표시한다.

그래프 [현월 예측 요금]

선택한 절감 대상 년/월의 가스사용량을 기준으로 현월 예측 요금을 그래프로 표시한다.

통보 설정

전월 실적 관리

전월 에너지 실적을 통보 한다. ( 디폴트 체크 )

당월 실적 관리

- 설정 목표가 입력한 % 초과시 고객에게 통보한다.

- 설정한 목표를 초과했을 시에 고객에게 통보한다. ( 디폴트 체크 )

- 주1회 에너지 절감 실적을 고객에게 통보한다.

- 주2회 에너지 절감 실적을 고객에게 통보한다.

설정 버튼

고객이 설정한 정보로 에너지 절감 실적을 통보 합니다.
