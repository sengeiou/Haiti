HEMS는 크게 3개의 패널로 구성되어 있다.

 

패널 명

설명

Header Panel

로그인 정보, 도움말 기능및 화면 테마 변경 기능 제공 패널

Tab Panel

Dashboard를 탭 방식으로 구성한 패널

Portlet Panel

각 Dashboard의 가젯들을 표시하기 위한 패널
