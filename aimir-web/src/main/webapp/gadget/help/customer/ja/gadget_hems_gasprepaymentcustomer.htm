AIMIR v4.0 Manual

요금제도가 선불인 가스 고객에 대해서, 충전 이력 조회및 잔액 통보 설정 기능을 제공합니다.

 

 

미니가젯

미니 가젯은 [충전이력] , [설정], [요금단가] 3개의 탭으로 구성되어 있습니다.
 

▪ 충전이력 : 마지막 충전이력및 다음 충전 예상일을 표시한다.
▪ 설정 : 잔액 통보 주기를 설정한다.
▪요금단가 : 선불 요금단가 테이블을 표시한다.

 

1) [충전이력] 탭

마지막 충전이력및 다음 충전 예상일을 표시합니다.
 

•Gauge챠트

    -.시작 값 : 0 (고정)

    - 종료 값 : 충전 후 잔액

    - 바늘 : 현재 잔액

    - 챠트 range 색깔 : 녹색 : 종료값의 50%, 노란색 : 종료값의 50% ~ 80%, 주황색 : 종료값의 80% ~
 

•마지막 충전 정보

    - 현재 잔액(원) : 현재 잔액을 표시한다.

    - 마지막 충전일 : 마지막 충전일을 표시한다.

    - 충전 후 잔액 : 충전 후 총 잔액을 표시한다.

    - 다음 충전 예상일 : 다음 충전 예상일을 표시한다.

 

 

 

2)  [설정] 탭

잔액 통보 주기를 설정합니다.

 

•잔액통보

    - 주기 : 일별 -> 일별로 잔액 주기를 통보 한다.

                  주별 -> 주별로 잔액 주기를 통보 한다.

    - 간격 :  통보 간격을 선택한다.

    - 요일 : 주기에서 주별 선택시 통보할 요일을 선택한다.

    - 시간 :  통보 시간을 선택합니다.

    - 통보값 : 통보값을 설정한다.
 

•Emergency Credit

    - Emergency Credit의 유효 기간을 표시한다.

 

 
 

 

3)  [요금단가] 탭

요금단가를 표시합니다.

2) [요금단가] 탭

요금단가및 충전금액으로 사용 예상일수를 계산하는 기능을 제공합니다.

 

 

•요금단가

      요금 단가표를 제공합니다. 요금 단가는 공급사에 따라서 달라질수 있습니다.

   

•충전 금액과 사용 가능 예상일수 계산

      충전하고자 하는 금액을 입력하시고 [계산]버튼을 클릭하면, 입력한 금액으로 예상 사용일수가 계산되어 표시됩니다.

      또한 충전금액당 사용 가능 전력량과 예상 사용일수를 디폴트로 표시합니다.
