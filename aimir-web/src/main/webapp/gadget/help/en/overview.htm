AIMIR v4.0 Manual

This article describes the new features of AIMIR 3.0 and explain each gadget including detailed explanation of every features and screen configuration.

 

AIMIR 3.0 login screen

 
This system is optimized for screen resolution of 1280x1024

 

Supported browsers are above IE 7.0, Google Chrome, Mozilla Firefox, Each browser screen may be slightly different with each browser feature

Flash Player 9.0 or later version of the browser must be installed since Flash Component of Adobe is implemented such as Flash Chart, etc.

English and Korean is supported.

 

Date, Number Format, Currency based on Locale of the system's construction is displayed
