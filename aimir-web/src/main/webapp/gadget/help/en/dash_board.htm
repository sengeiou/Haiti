AIMIR v4.0 Manual

Dashboard is the user interface (UI) to manage a variety of information in one place. AiMiR 3.0 provides specific features as a Gadget.

 

Multiple gadgets can be effectively managed in one place by using Dashboard.

 

The meaning of each tab is one dashboard. A user can use dashboards of maximum 8 but 4-5 dashboards are recommended.

When you click the tab on the screen, a dashboard would be displayed. Multiple Gadgets can be added on the dashboard( refer to Dashboard and Gadget Management Note), you can move a gadget within Dashboard.
