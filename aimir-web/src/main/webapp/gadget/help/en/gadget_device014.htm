1. 사용자 설명서

A.펌웨어 파일관리
1)배포할 파일을 등록하거나 수정 할 수 있는 기능을 제공 합니다.
a)펌웨어 등록 할 펌웨어 버전 정보 및 상세정보를 입력 할 수 있습니다.
b)등록할 펌웨어 정보를 입력후 Firmware Add버튼을 누릅니다.(펌웨어 배포가 되는 건 아님니다.)

 

1) 좌측프레임에는 장비, 밴더별로 펌웨어를 검색 할 수 있는 기능을 제공 합니다.
a)장비를 바로 찾을 수 있는 검색 기능을 제공 합니다.
b)우측 프레임에는 등록한 펌웨어 리스트가 출력 됩니다.

 

B. 펌웨어 배포

1)등록한 펌웨어 파일을 원하는 장비에 배포 할 수 있는 기능을 제공 합니다.
a)두개의 펌웨어를 선택 합니다.(체크박스클릭합니다.)

 

b) 두 개의 체크박스를 선택하면 하단 From, To 셀렉트 박스는 현재 설치된 펌웨어를 의미 하며 To는 업데이트할 펌웨어를 의미 합니다.(선택 박스에서 From - To 를 바꿀 수 있습니다.)

 

c)지역별로 설치된 장비가 나타나며 장비가 설치된 원하는 지역을 선택 합니다.(셀렉트박스클릭 합니다.)

 

d) 상위 내용을 모두 실행 하였다면 배포 버튼을 눌러 펌웨어 배포를 실행 합니다.

 

e) 배포후 H/W의 링크를 클릭하면 배포 결과를 간단하게 볼 수 있습니다.

 

f) 성공 여부를 판단하여 Retry 버튼을 클릭하여 배포 실패한 펌웨어를 재배포 할 수 있습니다.

 

C. 펌웨어 배포내역

1) 배포한 파일들의 결과 내역을 세부적으로 자세히 검색 확인 할 수 있습니다.

2) 지역 및 날짜등의 검색 조건으로 배포리스트를 검색 할 수 있습니다.

 

 

3) 검색 리스트에서 Trigger Id 링크를 클릭하면 세부 정보가 출력 됩니다.

 

4) 배포페이지와 마찬가지로 Retry 버튼을 클릭하여 배포 실패한 펌웨어를 재배포 할 수 있습니다.
