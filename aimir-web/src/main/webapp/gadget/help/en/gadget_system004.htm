AIMIR v4.0 Manual

Providing User account management, rights management as per customer, user group management services.

 

Mini Gadget

 

You can check the list of the users group including User ID, name, phone number, email and other basic information.

 

Max Gadget

 

Max Gadget can manage more detailed information by separating Group, the group user and login information.

 

▪Group Info

The user(admin) can manage the users by group. So the user can set options such as access, dashboard permission, gadget permission etc by user group. These options are applied to user group.

- Registration of New Group : You can add new user group by clicking " registration of new group" on the right.( to cancel the work please click the 'Close a registration page' button)

 

▪ Group User View

    - Registration User Group : If the user click this button, the user can add a new user in selected user group.

    - Registration New Group : If the user click this button, the user can add user group. If the user want to cancel, please click the 'Close a registration page' button.


 

▪ Login History

The user can check and search login history. ID, IP address, login / out, failed logins can be searched by a period.
