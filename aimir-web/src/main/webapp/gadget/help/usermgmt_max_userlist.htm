그룹별로 등록된 사용자와 사용자 상세 정보를 조회할 수 있다.

'그룹사용자목록'에서는 다음과 같은 기능을 갖는다.

1)그룹별 사용자 리스트 조회
2)새 사용자 추가 및 삭제
3)각 사용자 기본 정보 & 상세정보 확인 및 수정

 

최초화면에서는 사용자의 목록만을 보여주며, 리스트에서 특정 행을 클릭시 다음과 같이 화면 우측에 사용자 상세정보가 출력된다.

 

[Figure 2-3] 사용자 관리 맥스 가젯 - 그룹사용자목록
 

 

 

•사용자 상세 정보
-아이디
-사용자명
-별명
-Email 주소
-전화번호
-접속 제한
-접속 제한 사유

 

 

사용자 정보 수정

사용자의 상세정보 하단의 "변경" 버튼 클릭 시, 각 상세정보가 수정 가능한 형태로 표시되고

정보를 수정 후 "저장" 버튼을 클릭하면 사용자의 해당 정보가 수정된다.

"취소" 버튼을 클릭하면 기존의 DISABLE 상태로 돌아간다.

 

 

사용자 삭제

사용자 상세정보 하단의 "삭제" 버튼을 클릭하면 해당 사용자가 삭제된다.

 

 

사용자 추가

사용자 목록의 우측상단 "새 사용자 추가" 버튼을 클릭하면 [Figure 2-4] 와 같이 화면 우측의 사용자 상세정보 화면이 입력 가능한 상태의 화면으로 변경된다. 각 상세정보를 입력할 수 있는 형태로 FORM이 출력되며, 상세정보를 입력 후 "저장" 버튼을 클릭하면 새로운 사용자가 추가된다.

새로운 사용자가 추가되면 사용자 목록이 새로고침되어 추가된 사용자가 목록에 반영되어 나타난다.
