미니가젯
 

     

등록되어 있는 집중기, 모뎀, 미터, 고객, 계약의 개수를 Pie Chart와 Collumn Chart 로 한눈에 보여줍니다.

 

맥스가젯

 

장비등록 맥스 버전은 3개의 탭으로 이루어져 있습니다.

•일괄 등록 탭
•개별 등록 탭
•등록 이력 탭

 

•일괄 등록 탭

 

일괄 등록 탭은 아래와 같이 2개의 패널로 구성되어 있습니다.

1.장비 일괄 등록
2. 일괄 등록 장비 목록

 

 

일괄 등록 탭에서는 장비 타입을 선택하여 등록된 템플릿 파일을 다운로드 버튼을 사용하여 사용자 PC로 리스트를 다운로드 받을수 있습니다.

다운로드 받은 파일에 추가하고자 하는 장비 정보를 추가하여 등록 리스트를 작성할수 있습니다.

등록 리스트를 작성한 후에는 검색 버튼을 사용하여 사용자 PC 에서 파일의 위치를 지정할수 있고 등록 버튼을 사용하여 리스트를 등록합니다.

 

1. 장비 일괄 등록 방법

 a) 등록하고자 하는 장비의 템플릿 파일을 다운로드 받습니다.

 

 빨간색으로 네모쳐있는 곳에서 첫번째 SelectBox 는 자산의 큰 분류로 집중기, 모뎀, 미터, 고객 으로 분류됩니다.

            두번째 SelectBox는 첫번째 분류에 따라 각각 변화합니다.

첫번째 SelectBox

두번째 SelectBox

집중기

DCU, Indoor, Outdoor

모뎀

ACD, Converter, HMU, IEIU, IHD, MMIU, PLCIU, Unknown, ZBRepeater, ZEUMBus, ZEUPLS, ZMU, ZRU

미터

EnergyMeter, GasMeter, HeatMeter, VolumeCorrector, WaterMeter

고객

Customer, Contract

 

 b) 다운로드 받은 엑셀형식의 파일을 열어 등록하고자 하는 내용대로 작성해 줍니다.

 - 노란색으로 표시되어 있는 부분은 필수 값이므로 반드시 입력해야합니다.

 - 하단의 빨간색 박스의 Meter Format Description(EN)은 영어버전의 템플릿 작성 도움말이고 Meter Format Description(Ko)는 한글 버전 도움말입니다.

 

 c) 템플릿 파일 작성 완료 후 1번 패널의 일괄등록 파트에서 검색 버튼을 눌러 작성한 템플릿 파일을 읽어옵니다.

 

 - 파일을 제대로 읽어오게 되면 2번 패널처럼 맨 앞에 Data Status 필드에 Success라고 입력되고 잘못된 형식이나,

   필수 값을 입력하지 않게 되면 Failure 라는 값이 입력되게 됩니다.

 d) 파일을 불러온 후 검색 버튼 옆의 등록 버튼을 클릭하게 되면 등록을 시도한 장비의 전체 개수와 성공/실패 개수를 확인할 수 있습니다.

 

•개별 등록 탭

개별 등록 탭에서는 장비의 세부 정보를 입력하여 개별적으로 장비를 등록할수 있습니다.

이때 통합 NMS 내에서 집중기,모뎀,미터의 아이디의 유일성을 유지하기 위하여 등록시에 아이디 중복 체크를 실행할수 있습니다.

이때 입력 하는 장비의 세부 정보를 설치되어 있는 집중기,모뎀,미터를 바탕으로 입력해야 합니다.

 

•등록 이력 탭

등록 이력 탭에서는 장비 타입,아이디,제공자,모델,등록 타입,등록 결과 등 장비를 등록 한 이력에 대한 내용을 제공하고 위의 상세 정보에 대하여 검색을 통해 해당 리스트를 표현합니다.

자산

자산의 종류를 표시합니다. 자산 종류에는 DCU, Meter, Modem, Customer가 존재합니다.

자산 타입

자산의 세부 타입을 표시합니다.

- DCU의 세부 타입 : DCU, Indoor, Outdoor

- Modem의 세부 타입 : ACD, Converter, HMU, IEIU, IHD, MMIU, PLCIU, ZBRepeater, ZEUMBus, ZEUPLs, ZMU, ZRU

- Meter의 세부 타입 : EnergyMeter, GasMeter, HeatMeter, VolumeCorrector, WaterMeter

- Customer의 세부 타입 : Customer, Contract

자산 아이디

자산 아이디를 표시합니다.

제조사

제조사 이름을 표시합니다.

모델

모델의 이름을 표시합니다.

등록 타입

자산를 등록한 방법을 표시합니다.

- Auto Reg : 자동으로 등록된 경우.

- Bulk Registration : Bulk Registration 탭에서 엑셀파일을 이용해 등록한 경우.

- External System Registration : Meter Management나 Modem Management, Concentrator Management 등 외부 시스템에서 등록한 경우

- Single Registration : Single Registration탭에서 자산를 등록한 경우.

등록 결과

등록의 성공여부를 Success / Failure 로 표시합니다.

설치 일시

등록을 시도한 날짜를 표시합니다.
