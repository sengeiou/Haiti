전기의 전력 품질(Voltage, Current)  , Voltage Level, Voltage Imbalance 정보를 제공합니다.

 

미니가젯

 

전기 미터의 경우 검침 주기 별 전력 품질데이터를 의미합니다.

전기 미터의 경우에만 Power Quality데이터가 존재합니다.

 

 

•일별, 주별, 월별 검색을 지원합니다.
•Voltage Unbalance : 평균과 최대 최소의 오차가 입력한 값을 넘어가는 경우
•위상각 역방향 불균형 : 기간의 Voltage Angle값 중에 부호가 마이너스인 값이 있는 경우

   Voltage Angle의 경우 phase 3개중 하나라도 마이너스 값이 있으면 카운트가 1개씩 올라감.

 

※ Voltage Phase 별(Voltage(A), Voltage(B), Voltage(C)) 최대 최소 편차 % 구하는 공식 : 100*최대-최소/평균

 cf) Voltage Phase 별 최대 최소 편차는 Voltage Variation시간대별 각 phase별 전압 불균형를 의미합니다.

 

맥스가젯

 

Power Quality 맥스 버전은 2가지 탭으로 구성됩니다.

•[Voltage Levels] 탭
•[power Instrument] 탭

 

•[Voltage Levels] 탭

 

•Period, Weekly, Monthly 세가지 타입으로 기간을 조회할 수 있습니다.
•장비 타입 All, MCU, Modem, Meter 중에서 선택해 데이터를 조회할 수 있습니다.
•최대/ 최소 오차(%)를 입력해 그 오차의 범위에 해당하는 데이터를 조회 할 수 있습니다.
•

  - Date : 데이터가 올라온 날짜를 표시합니다.

  - 장비 타입 : 장비의 타입, 타입의 종류는 집중기, 모뎀, 미터 3가지가 있습니다.

  - 장비아이디 : 장비의 ID

 - 고객명 : 고객의 이름

•검색된 장비의 최소, 최대, 평균 값을 각각 Voltage Phase 별(Voltage(A), Voltage(B), Voltage(C))로 출력해 줍니다.
•평균과 최대 최소의 오차가 입력한 값을 넘어가는 경우인 Voltage Unbalance 상태의 장비를 검색합니다.
•최대와 최소의 차이가 클수록 Power의 품질은 떨어집니다.

 

 

•[Power Instrument] 탭

 

•Period, Weekly, Monthly 세가지 타입으로 기간을 조회할 수 있습니다.
•장비 타입 All, MCU, Modem, Meter 중에서 선택해 데이터를 조회할 수 있습니다.
•최대/ 최소 오차(%)를 입력해 그 오차의 범위에 해당하는 데이터를 조회할 수 있습니다.
•Reverse phase Angle 항목에 체크해 위상각 역방향인 데이터를 조회할 수 있습니다.

 

  Power Instrument 탭은 크게 2개의 패널로 구성되어 있습니다.

  1. Phase별 Voltage 정보

  2. 선택된 항목에 대한 상세 정보

 

 

 

  1. Phase별 Voltage 정보

  - Date : 데이터가 올라온 날짜를 표시합니다.

  - Equip. Type : 장비의 타입, 타입의 종류는 MCU, Modem, Meter 3가지가 있습니다.

  - Equip. ID : 장비의 ID

  - Last Metering Date : 마지막 검침 일자

  - Contract No. : 계약 번호

  - Customer Name : 고객의 이름

 

  각 A, B, C 각 상별로 입력된 모든 전력량을 표시합니다.

  - Voltage : Phase A, Phase B, Phase C의 전압값을 표시합니다.

  - Current : 현재의 Phase A, Phase B, Phase C의 전류량을 표시합니다.

  - Line Valtage : AB, CA, BC 로 세분화 되어 있습니다. AB의 경우 Phase A와 Phase B의 전압차이를 의미합니다.

 

  2. 선택된 항목에 대한 상세 정보

•1번 패널에서 선택한 항목에 대한 상세정보가 표시됩니다.
•전력의 품질의 구성하는 종류(Type View)에 대해 상세하게 선택해서 Phase 별 데이터를 볼 수 있습니다.
•Type View 종류

  a) Voltage Per Phase

  b) Current Per Phase

  c) Voltage Angle Per Phase

  d) Current Angle Per Phase

  e) Power Factor Per Phase

   f) Voltage THD Per Phase

  g) Current THD Per Phase

  h) TDD Per Phase

   i) Distortion Power Factor Per Phase

   j) Momentary Interval Distortion kVA Demand

  k) Fundamental (1st harmonic) Voltage Magnitude

   l) Fundamental (1st harmonic) Current Magnitude

 m) Voltage 2nd Harmonic Magnitude

  n) Current 2nd Harmonic Magnitude

  o) 2nd Harmonic Voltage %

  p) Harmonic Current (sum of 2nd through 15th)

  q) Line Frequency

   r) System Power Factor Angle
