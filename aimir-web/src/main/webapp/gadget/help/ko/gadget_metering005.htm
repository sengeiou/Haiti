검침실패(전기)

 Top  Previous  Next

검침 실패 고객들에 대한 통계 및 실패 원인, 재 검침 서비스 등의 정보를 제공합니다.

 

미니가젯

 

검침실패 미니 버전은 2가지 패널로 구성됩니다.

1. 일별, 기간별, 주별, 월별 데이터 조회

2. 검침 실패율 표시 차트 및 그리드

 

 

1. 일별, 기간, 주별, 월별 데이터 조회

일별, 기간, 주별, 월별 데이터를 조회할 수 있습니다.

 

2. 검침 성공률 표시 차트 및 그리드

그리드 버전을 선택 시 왼쪽 화면이 차트 버전을 선택 시 오른쪽 화면이 나타납니다.

•그리드 버전에서 검침실패율의 숫자는 검침 실패율(검침 실패 횟수/전체 검침 횟수)를 의미합니다.
•차트 버전에서 마우스를 막대에 가져다 대면 해당 막대그래프에 대한 정보가 표시됩니다.

 

맥스가젯

 

검침실패 미니 버전은 2가지 패널로 구성됩니다.

1. 일별, 기간별, 주별, 월별 데이터 조회

2. 엑셀, 상세보기 버튼

3. 검침실패율 차트

4. 실패원인 차트

5. 검침 실패 목록

 

 

1. 일별, 기간별, 주별, 월별 데이터 조회

일별, 기간, 주별, 월별 데이터를 조회할 수 있습니다.

 

2. 엑셀, 상세보기 버튼

•[Excel] 버튼

검침 실패 목록을 엑셀파일로 제공합니다.

•[Details] 버튼

Location, Contract No. Meter ID, DCU ID 별로 상세 조회가 가능합니다.

검침 실패한 미터에 대한 상세정보를 볼 수 있고 실패한 미터를 재검침 할 수 있는 기능이 있습니다.

재검침하고자 하는 계약목록을 체크하고 Excute버튼을 클릭하면 재검침을 시작합니다.

•[Back] 버튼

Detail버튼을 클릭하면 이 버튼은 Back 버튼으로 변경됩니다. 이따 Back 버튼을 누르면 바로 전 화면인 데이터 조회 화면으로 되돌아 가게됩니다.

3. 검침실패율 차트

지역에 따라 검침 실패율을 차트로 보여줍니다.

차트를 클릭하게 되면 하위 지역이 존재할 경우 클릭한 지역의 하위 지역에 대한 내용이 표시됩니다.

x 축 : 지역을 의미합니다.

y 축 : % 로 검침이 실패한 비율을 의미합니다.

 

4. 실패원인 차트

실패 원인1, 실패 원인2, 기타의 내용을 명시하고 그 비율을 차트로 보여줍니다.

 

5. 검침 실패 목록

•검침에 실패한 목록을 지역에 따라 표로 보여줍니다.
•그리드에 표시된 항목을 클릭하면 Details 버튼을 누른 것과 같은 화면으로 이동합니다. 클릭한 항목에 해당하는 미터들의 상세정보를 표시하게 됩니다.

지역

공급 지역을 의미합니다.

검침실패율

실패율을 의미합니다.

전체 건수

총 미터의 개수를 의미합니다.

실패 건수

검침에 실패한 미터의 개수를 의미합니다.

실패 원인 1

검침에 실패한 미터 중 4번 패널의 하단에서 정의한 실패 원인중 원인1 해당하는 미터들의 개수를 의미합니다.

실패 원인 2

검침에 실패한 미터 중 4번 패널의 하단에서 정의한 실패 원인중 원인 2에 해당하는 미터들의 개수를 의미합니다.

기타

검침에 실패한 미터 중 4번 패널의 하단에서 정의한 실패 원인중 원인1, 원인2 에 해당하는 않는 미터들의 개수를 의미합니다.
