제조사 모델 관리

 Top 

시스템에서 사용하는 장비의 제조사, 모델 정보 및 특정 모델에 특화된 정보조회, 모델 사진 등의 서비스를 제공

 

미니 가젯

현재 시스템에 등록되어 있는 장비 목록을 간단하게 확인 할 수 있습니다. 장비별, 제조사별로 정렬할 수 있습니다.
장비별 정렬 : 집중기, 모뎀, 미터 등 타입별로 장비 목록을 보여줍니다.
제조사별 정렬 : 장비를 제작한 제조사별 장비 목록을 보여줍니다. 목록에서 등록된 장비의 모델명을 확인 할 수 있습니다.

 

 

맥스 가젯

각 장비에 대한 상세정보를 확인 할 수 있습니다. 왼쪽 장비목록은 미니 가젯과 동일하게 장비별, 제조사별로 정렬 할 수 있습니다.

목록에서 제조사를 선택시 오른쪽에 제조사 상세 정보가, 장비 선택 시는 오른쪽에 해당 장비에 대한 상세 정보가 표시됩니다.

 

가젯 오른편에서는 제조사, 장비, 설정을 추가할 수 있습니다.

•제조사 추가 : '제조사 추가' 버튼을 클릭하면 새로운 제조사를 등록할 수 있습니다. 제조사명, 고유 코드, 제조사의 주소, 설명을 입력할 수 있습니다. *고유 코드는 별도의 지정된 코드를 입력해야 합니다. 자세한 사항은 누리텔레콤에 문의해 주시기 바랍니다.
•장비 추가 : 새로운 장비 정보를 추가 할 수 있습니다.
•이미지 수정 : 하단의 '이미지 파일 찾기' 버튼을 클릭하면 추가하려는 장비의 사진을 추가 또는 수정할 수 있습니다.
•제조사 : 추가하려는 장비의 제조사 명을 선택합니다. 제조사명을 모를 경우 하단의 Unknown 을 선택합니다.
•고유 코드 : 모델의 고유 코드를 입력합니다. 코드값은 코드 관리 가젯에서 관리합니다. 코드값을 모를 경우 누리텔레콤에 문의하시기 바랍니다.
•모델 : 장비의 고유 모델명을 입력합니다.
•타입 : 장비의 타입을 선택합니다. Indoor, Outdoor, DCU, ZRU, ZEUPLS 등 해당하는 타입을 선택합니다. 없을 경우 하단의 '기타'를 선택합니다.
