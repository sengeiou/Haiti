 Top  Previous  Next

대시보드는 다양한 정보를 한 곳에 모아 관리하고 사용할 수 있도록 하는 사용자 인터페이스(UI) 단위입니다.
 
AiMiR 3.0은 하나의 특화된 기능을 하나의 가젯(Gadget)으로 제공하고 있습니다.
대시보드를 이용하면 여러 개의 가젯을 하나의 대시보드에 모아 한 곳에서 관리할 수 있습니다.
기능별로 또는 자주 사용하는 가젯을 대시보드 별로 분류하면 효율적으로 시스템을 관리할 수 있습니다.

각 탭은 하나의 대시보드를 뜻하며 사용자는 최대 8개의 대시보드를 사용할 수 있지만 브라우저의 특성상 4-5개의 대시보드 사용을 추천합니다.
탭을 클릭하면 해당하는 대시보드를 화면에 표시합니다.

사용자는 대시보드에 여러 개의 가젯을 추가하고(Dashboard and Gadget Management 참고) 대시보드내에서  가젯의 위치를 이동 시킬 수 있습니다.
보고싶지 않은 가젯은 비활성화 시킬 수 있습니다.
 
하나의 대시보드당 가젯은 최대 16개까지 추가할 수 있으나 가젯의 로딩 속도가 느려지므로 그 이하로 사용할 것을 권장합니다.

AiMiR 3.0은 사용자별 대시보드를 제공함으로써 사용자마다 각각 다른 구성의 대시보드를 만들 수 있고 사용자가 가젯의 위치를 변경하면 위치를 저장하여 대시보드의 레이아웃이 그대로 유지됩니다.

대시보드에서 특정 가젯을 최대화 하면 대시 보드안에 해당 가젯이 최대화 되며 다시 해당 가젯을 최소화 하면 원래 화면 배치로 돌아오게 됩니다.
