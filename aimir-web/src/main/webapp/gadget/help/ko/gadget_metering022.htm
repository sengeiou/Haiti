맥스가젯

 

SLA  맥스 버전은 2개의 탭으로 구성되어 있습니다.

•SLA 전체 분석 탭
•검침성공률 분석 탭

 

•SLA 전체 분석 탭

 

SLA 전체 분석 탭은 3개의 패널로 구성되어 있습니다.

1. 기간 설정

2. SLA(%)에 대한 차트

3. SLA(%)에 대한 상세 목록

 

 

 

1. 기간 설정

기간, 주별, 월별 조회가 가능합니다.

 

2. SLA(%)에 대한 차트

SLA미터의 개수, Delivered된 미터의 개수, SLA 성공률을 차트로 나타냅니다.

SLA미터 : Contract의 적용일자가 오늘보다 과거인 미터의 수

Delivered : 연계시스템으로 검침정보가 전달된 미터를 의미합니다.

 

3. SLA(%)에 대한 상세 목록

설치된 총 미터 개수, AIMIR에서 수집한 총 미터 수, SLA 미터 수, 유틸리티사로 전달된 미터 수, SLA(%) 성공율 필드에 맞춰 리스트를 출력합니다.

설치된 총 미터 개수

등록된 모든 미터 수

AIMIR에서 수집한 총 미터 수

오늘 검침에 성공한 미터 수

SLA 미터 수

Contract의 적용일자 오늘보다 과거인 미터의 수

유틸리티사로 전달된 미터 수

연계 시스템으로 검침정보가 전달된 미터의 수

 

 

•검침성공률 분석 탭

 

검침성공률 분석 탭은 3개의 패널로 구성되어 있습니다.

1. 기간 설정

2. 누락 건수에 관한 정보

3. 누락 원인에 대한 정보

 

 

1. 기간 설정

기간, 주별, 월별 조회가 가능합니다.

 

2. 누락 건수에 관한 정보

오른쪽 그리드에 통신 실패 날짜와 그 날에 몇 번의 누락이 있는 미터가 몇개가 존재하는지, 실패율이 표시됩니다.

- 통신 실패 일수 : 통신에 실패한 날짜를 표시합니다.

- Count : 그 날에 몇 번의 누락이 존재하는지 표시합니다.

- % : 성공률을 표시하는 것이기 때문에 Count가 0이면 %는 100%가 됩니다

•그리드의 출력된 정보를 한 눈에 볼 수 있도록 왼쪽에 차트로 표시되어집니다.

- 왼쪽 y축 : 몇번의 검침 누락이 있었는지 개수

- 오른쪽 y축 : 성공률

- 축 : 실패한 날짜

 

3. 누락 원인에 대한 정보

2번 패널의 그리드에서 항목을 하나 선택하면 3번 패널에 선택한 항목에 대한 상세정보가 표시됩니다.

실패한 건수에 대한 미터의 정보와 최종 통신일시, 통신 장비 아이디, 계약번호가 오른쪽 그리드에 표시되고 왼쪽에 미터가 누락된 원인이 차트로 표현됩니다.

차트는 실패원인에 대해 색깔별로 구분하여 선택한 날짜의 누락에 대해 실패원인의 비율을 표시합니다.

•실패원인의 종류

Integration Failed

DCU Connection Error

Modem Connection Error

GPRS Problem

DCU Acceptable Limits

Meter Abnormal
