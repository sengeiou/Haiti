사용자 관리

사용자 계정관리, 사용자 별 권한 관리, 사용자 그룹관리 서비스 제공

 

미니 가젯

사용자 그룹별로 사용자의 목록을 보여줍니다. 사용자의 아이디, 이름, 전화번호, 이메일등의 기본정보를 확인할 수 있습니다. 사용자그룹 리스트 박스를 클릭하면 다른 사용자그룹을 선택할 수 있습니다.

 

맥스 가젯
맥스 가젯은 사용자 관리를 위해 더욱 상세한 정보를 관리할 수 있습니다. 이를 위해 그룹 정보, 그룹 사용자 정보, 로그인 이력으로 세분화한 정보를 제공합니다.  각 기능은 탭으로 구분이 되었으며 왼쪽 상단에 있는 리스트박스에서 그룹을 선택하면 해당 그룹의 정보를 확인하고 설정할 수 있습니다.

 

1.그룹 정보
AiMiR 3.0은 사용자 그룹을 생성하여 비슷한 종류의 사용자들을 하나의 그룹으로 묶어서 관리할 수 있습니다. 그룹 정보 탭에서는 그룹별로 접속 허용, 대시보드 권한, 검침데이터 및 시스템정보 읽기/쓰기 권한, Ondemand 명령의 권한 등을 설정 할 수 있습니다. 설정된 정보는 그룹에 속한 모든 사용자에게 적용이 됩니다.
 
그룹 정보에서는 가젯의 허용 유무 설정도 할 수 있습니다. 특정 그룹에 전체 가젯 중 전체 또는 일부만 허용을 할 수 있어 사용자의 종류별로 가젯의 접근을 제한할 수 있습니다.

 
-새 그룹 등록 : 우측 상단의 '등록 새그룹' 을 클릭하면 새로운 사용자 그룹을 생성할 수 있습니다.("등록화면 닫기"를 클릭하면 작업을 취소할 수 있습니다)
 
•공급사 선택 : 목록에서 등록된 공급사 중 해당하는 곳을 선택합니다.
•그룹명 : 새로 등록할 사용자 그룹의 이름을 입력합니다. '중복 확인' 버튼을 클릭하여 사용할 수 있는 이름인지 확인합니다.
•접속허용여부
•대시보드 권한
•검침데이터 : 검침데이터에 대한 접근 권한을 설정합니다.
읽기 : 검침데이터를 확인만 할 수 있습니다.
쓰기 : 수정이 가능한 자료에 대해서는 데이터를 변경할 수 있는 권한을 제공합니다. 관리자에게만 추천되는 권한입니다.
명령 : Ondemand, UnitScanning 과 같은 원격 명령 실행에 대한 권한을 설정합니다. '명령' 권한은 면 '읽기/쓰기' 권한도 포함합니다.
*CDMA/GPRS 등의 통신 방식을 사용할 경우 원격 명령 수행 시 별도의 데이터 요금이 발생할 수 있으므로 주의가 필요합니다.
•시스템 정보
읽기 : 시스템 정보를 확인할 수 있습니다.
쓰기 : 시스템 정보를 수정할 수 있는 권한을 제공합니다.
명령 : 원격으로 시스템 정보를 확인할 수 있는 항목에 대해 원격 명령 권한을 제공합니다.
*CDMA/GPRS 등의 통신 방식을 사용할 경우 원격 명령 수행 시 별도의 데이터 요금이 발생할 수 있으므로 주의가 필요합니다.
•설명 : 등록할 사용자 그룹에 대한 설명을 입력합니다.
•명령 : 앞에서 '명령' 권한을 설정하였다면 모뎀, 집중기, 미터 별로 허용할 '명령'을 선택할 수 있습니다.
•가젯 설정
새로 생성하는 사용자 그룹에 허용할 가젯을 선택할 수 있습니다. 오른쪽의 전체 가젯 목록에서 허용할 가젯을 선택한 후 왼쪽 화살표를 클릭하면 왼쪽의 허용된 가젯 목록에 추가됩니다. 취소하려면 허용된 가젯 목록에서 가젯 선택 후 오른쪽 화살표를 클릭하십시오.
2.그룹 사용자 정보
 
사용자 목록 : 그룹에 속한 사용자 목록을 볼 수 있습니다.
그룹 사용자 등록 : 그룹에 새로운 사용자를 추가 할 수 있습니다.

그룹 사용자 등록 : 화면 오른편의 '그룹 사용자 등록' 폼을 입력하여 사용자 그룹에 새로운 사용자를 추가할 수 있습니다.
•아이디 : 추가할 사용자의 아이디를 입력합니다. 이 아이디는 로그인시 사용됩니다. 입력후에는 '중복확인' 버튼을 클릭하여 사용 가능 여부를 확인합니다.
•사용자명 : 사용자의 이름을 입력합니다.
•별명 : 사용자의 별명을 입력합니다.
•비밀번호/비밀번호 확인 : 비밀번호를 입력합니다. 로그인 시 사용됩니다.
•Email 주소 : 사용자의 Email 주소를 입력합니다.
•전화번호 : 사용자의 번화 번호를 입력합니다.
•접속 제한 : 'YES' 를 선택할 경우 해당 사용자의 로그인을 허용하지 않습니다.
•접속 제한 사유 : 접속을 제한하는 사용자의 경우 이에 대한 설명을 입력할 수 있습니다.

 

3.로그인 이력
AiMiR 에 접속했던 사용자들의 접속 내역을 확인할 수 있습니다. 아이디, IP주소, 로그인/아웃, 로그인 실패를 기간별로 검색 할 수 있습니다.
