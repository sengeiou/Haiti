이 문서는 AIMIR 3.0 버전의 신규 기능 및 각 가젯 들에 대해 기술하며 각 세부기능에 대한 설명, 화면 구성에 대한 내용을 설명하고 있습니다.

 

 

AIMIR 3.0 로그인 화면

 

 

해당 시스템의 화면은 1280*1024 해상도(resolution)에 최적화되어 있습니다.

 

웹 표준을 지향하며, 지원 브라우저는 IE 7.0 이상, Google Chrome, Mozilla Firefox 입니다.

브라우저의 특성상 브라우저 별 화면이 조금 다를 수 있습니다.

해당 시스템은 또한 Flash Chart 등 Adobe사의 Flash Component 를 구현하였으므로 브라우저에 Flash Player 9.0 이상의 버전이 설치 되어있어야 합니다.

지원 언어는 현재 영문, 국문을 지원합니다.

날짜, 숫자 포맷, Currency는 시스템의 공사 정보의 Locale을 기준으로 하여 표시합니다.
