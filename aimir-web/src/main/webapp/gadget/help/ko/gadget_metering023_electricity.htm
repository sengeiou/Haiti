VEE(전기)

 Top  Previous  Next

검침데이터에 대한 Valication Check, Editing Estimation을 할 수 있는 기능을 제공하며 Editing Estimation을 했던 이력등을 조회할 수 있는 기능을 제공합니다.

 

 

□ Mini Gadget

 

   미니 가젯에서는 검침 데이터 Editing 및 Estimation 기능은 제공하지 않으며 조회 기능만 제공합니다.

 

   검침값 유효성 관리(전기) 미니버전은 아래와 같이 1가지 패널로 구성됩니다.

   1. 데이터 유효성 검사/검침데이터 변경 이력 탭

 

   1. 데이터 유효성 검사/검침데이터 변경 이력 탭

   이 패널은 [데이터 유효성 검사], [검침데이터 변경 이력]  2개의 탭으로 구성됩니다.

 

•데이터 유효성 검사
•검침데이터 변경 이력

 

 

 

•데이터 유효성 검사

  데이터 유효성 검사 패널은 다음과 같이 2가지 패널로 구성됩니다.

   1. 주기 선택 및 조회

   2. 카테고리별 유효성 조회 리스트

 

   1 주기 선택 및 조회

•일/주/월/요일 별 콤보 선택이 가능합니다.
•선택한 주기에 따라 2번패널의 리스트가  표현됩니다.

 

    2. 카테고리별 유효성 조회 리스트

•검침값 유효성 조건 별 해당하는 포인터별 개수 조회가 가능합니다.
•각 항목별 상세 설명은 다음과 같습니다.

        - Negative Consumption : 검침데이터 값이 0보다 작은 경우

        - Zero Consumption : 검침데이터 값이 0인 경우

        - Data Gaps : 검침이 중간에 실패해서 값이 없는경우

        - Out of Threshold : 임계치 값을 초과한 경우

        - Abnormal Power Factor : 비정상적인 Power Factor

        - Abnormal Meter Time : 미터의 시간이 비정상적인 경우

        - Abnormal Meter Id : 미터의 아이디가 비정상적인 경우

        - Abnormal Low Usage : 비정상적으로 사용량이 낮은 경우

        - Abnormal Low Demand : 비정상적으로 수요가 낮은 경우

        - Abnormal Load Factor : 비정상적인 Load Factor

        - Abnormal High Usage : 비정상적으로 사용량이 높은 경우

        - Abnormal High Demand :비정상적으로 수요가 많은 경우

 

•검침데이터 변경 이력

   검침값 변경에 따른 이력 조회가 가능합니다.

   검침데이터 변경 이력 패널은 다음과 같이 2가지 패널로 구성됩니다.

   1. 주기 선택 및 조회

   2. 이력 조회 리스트

 

   

 

   1 주기 선택 및 조회

•일/주/월/요일 별 콤보 선택이 가능합니다.
•조회는 "조회"버튼 클릭시 가능합니다.
•선택한 주기에 따라 2번패널의 리스트가  표현됩니다.

     

   2 카테고리별 유효성 조회 리스트

•1번 패널에서 선택한 주기 및 조건에 해당하는 검침데이터의 변경이력을 보여줍니다.
•카테고리, 개수 이력 조회가 가능합니다.

 

 

맥스 가젯

 

   검침값 유효성 관리(전기) 맥스 버전은 3 개의 탭으로 구성됩니다.

•데이터 유효성 검사
•검침데이터 변경 이력
• VEE 조건 설정

 

 

 

•데이터 유효성 검사

  검침데이터의 조회 및 Editing /Estimation 기능을 제공합니다.

  데이터 유효성 검사 패널은 다음과 같이 3가지 패널로 구성됩니다.

  1. 주기 선택 및 조회

  2. 카테고리별 유효성 조회 리스트

  3. 주기별 검침데이터

 

  1 주기 선택 및 조회

•일/주/월/요일 별 콤보 선택이 가능합니다.
•검침데이터 유효성 별 콤보 선택이 가능합니다.

        - Negative Consumption : 검침데이터 값이 0보다 작은 경우

        - Zero Consumption : 검침데이터 값이 0인 경우

        - Data Gaps : 검침이 중간에 실패해서 값이 없는경우

        - Out of Threshold : 임계치 값을 초과한 경우

        - Abnormal Power Factor : 비정상적인 Power Factor

        - Abnormal Meter Time : 미터의 시간이 비정상적인 경우

        - Abnormal Meter Id : 미터의 아이디가 비정상적인 경우

        - Abnormal Low Usage : 비정상적으로 사용량이 낮은 경우

        - Abnormal Low Demand : 비정상적으로 수요가 낮은 경우

        - Abnormal Load Factor : 비정상적인 Load Factor

        - Abnormal High Usage : 비정상적으로 사용량이 높은 경우

        - Abnormal High Demand :비정상적으로 수요가 많은 경우

•조회는 "조회"버튼 클릭시 가능합니다.
•선택한 주기에 따라 2번패널의 리스트가  표현됩니다.

 

   2. 카테고리별 유효성 조회 리스트

•1번 패널 검색에 따라 2번 조회 리스트가 출력됩니다.
• Editing&Estimation을 하기 위해 선택이 가능합니다.
•선택에 따라 3번 패널에 검침데이터가 주기별 표현됩니다.

 

   3 주기별 검침데이터

•2번 선택에 따른 검침데이터 조회가 가능합니다.
•Editing&Estimation 기능을 제고합니다.

Apply

realData 뒤에 표시되는 검침데이터를 더블클릭하면 수정이 가능합니다.

이때 수정후 이 데이터를 적용시키려 할때 Apply 버튼을 눌러 적용시킵니다.

Metering Day

검침된 날짜를 표시합니다.

channel

채널정보를 표시합니다.

dst

서머타임에 대한 정보를 표시합니다.

realData

실제 데이터인지 여부를 Y/N 으로 표시합니다.

 

realData 필드 뒤 부터는 lp_interval에 따른 검침데이터가 표시됩니다.

 

•Editing은 다음과 같은 순서로 진행합니다.

   a) 변경하고자 하는 검침데이터의 컬럼 더블클릭후 입력합니다.

   b) 입력후 Apply버튼을 클릭하여 데이터를 저장합니다.

 

•Estimation은 다음과 같은 순서로 진행합니다.

    a) 해당 미터별 검침데이터 날짜에 컬럼을 선택합니다.

    b) 선택되어진 컬럼은 파란색으로 표시됩니다.

    c) 주기를 선택 후 테이블 우측 상단의 AutoEstimation 버튼을 클릭합니다.

    d) 다음과 같은 팝업 윈도우가 나타납니다.

   

     e) 전일 콤보를 클릭합니다.

     f) 다음과 같은 리스트가 보여집니다.

         

           이는 Estimation을 할 기준 데이터 일자를 선택하는 것입니다.

      g) 동일구간 콤보를 클릭합니다.

      h) 다음과 같은 리스트가 보여집니다.

         

            이는 데이터 값을 가져올 설정을 선택한는 것입니다.

      i) 미리보기를 통해 확인후 적용 버튼을클릭합니다.

 

•검침데이터 변경 이력
•1번 탭에서 변경된 검침 데이터 조회가 가능합니다.
•변경된 검침값의 범위, 변경된 검침값의 종류  계약번호, 사용자 아이딜 로 조회가 가능합니다.

    - 변경된 검침값의 범위 : LoadProfile, Day, Month

    - 변경된 검침값의 종류 : Autoatic Estimated, Individual Edited, RuleBased Estimated, User Defined Estimated, Verified
