집중기 관리

공사에서 관리하는 집중기 목록 및 상세 정보,집중기 별 검침스케줄 설정,집중기 설치 위치 정보,집중기 설정 및 제어 등의 기능을 제공

 

•맥스 가젯

- Basic Information

집중기 관리 화면에서는 기본 조회를 통해 등록된 집중기 리스트를 나타낸다.그리고 집중기 타입,위치,SW 버젼,HW 버젼,설치 일자,프로토콜 타입,집중기 아이디 등의 세부 정보를 입력하여 나타내고자 하는 집중기를 검색할수 있다.리스트에서 확인 하고자 하는 집중기를 선택하면 화면 아랫부분에 선택한 집중기의 정보가 나타난다.집중기 정보의 기본 정보 탭에서는 집중기 설치 사진 및 집중기 아이디,집중기 타입,제공자,모델,설치 일시 등의 세부정보가 나타난다.표현된 세부정보에 대해서는 수정이 가능하다.

 

- Location(Map) Info

Location(Map) info 탭에서는 설치된 집중기의 Map 정보를 Google Map 을 이용하여 표현한다.또한 집중기에 연결된 장비 리스트에 대해서도 확인 할수 있다.

 

- Schedule

스케쥴 탭에서는 집중기의 검침 스케쥴을 저장 할수 있다.스케쥴의 종류로써는 검침 스케쥴,이벤트 로그 스케쥴,미터 타임 동기화 스케쥴,검침 데이타 업로드 스케쥴,데이타 복구 스케쥴 등의 세부 스케쥴 설정을 수정할수 있다.

 

- History

이력 탭에서는 해당 집중기의 일반 로그,경고 이력,명령어 수행 이력등을 기간별로 조회 할수 있다.
