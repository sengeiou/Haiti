코드 관리

 Top 

시스템에서 관리하는 코드의 정의 및 코드 목록, 코드 정보 추가/수정 삭제 기능을 제공합니다.

 

미니 가젯

시스템에 등록된 코드 정보를 간단히 검색 및 추가/삭제 하는 기능을 제공합니다.

검색 : 검색창에 검색하려는 코드명 또는 코드값을 입력 후 검색버튼을 클릭하면 해당되는 항목은 빨간색으로 표시됩니다.

추가 : 새로운 코드 항목을 추가할 수 있습니다. 코드 목록중 한 항목을 선택하고 추가 버튼을 클릭하면 자식 노드에 빈 항목이 한 줄 추가됩니다. 코드명, 코드값, 설명을 입력합니다. 저장하기 전 추가 상태에서는 새로운 항목뿐 아니라 기존의 다른 항목도 수정을 할 수 있습니다.

맥스 가젯

코드 관리 가젯은 맥스 가젯과 미니 가젯의 기능이 동일합니다. 맥스 가젯은 넓은 화면에서 코드 정보를 좀 더 편하게 관리할 수 있습니다.
