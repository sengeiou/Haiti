가젯

가젯은 Google사의 iGoogle을 모티브로 하여 비활성모드, 축약화면 (Mini Gadget), 최대화면 (Max Gadget)의 형태를 지니며 각 가젯별로 가젯 비활성화, 활성화, 도움말, 새로 고침, 최대화, 최소화, 가젯 삭제 등의 컴포넌트를 가지고 있습니다.

 

 

①

축소

가젯을 타이틀바만 보이도록 축소합니다.  아무런 내용이 보이지 않기 때문에 기능을 비활성화 시킵니다.

②

도움말

해당 가젯의 도움말을 보여줍니다.

③

리로드

가젯에 표시된 내용을 다시 로드합니다.

④

최대화/최소화

미니가젯<->맥스가젯으로 전환합니다.

⑤

삭제

가젯을 대시보드에서 삭제합니다.(삭제한 가젯은 설정에서 다시 추가할 수 있습니다)

 

 

일반적으로 미니 가젯에서는 축약된 정보나 요약된 그래프등을 보여주며 맥스가젯에서는 상세정보 등을 보여줍니다.

 

AiMiR 3.0은 하나의 가젯에 하나의 특화된 기능을 제공합니다.

예를 들어 검침 실패 가젯의 경우 검침실패와 관련하여 일별, 기간별, 주별, 월별 검침실패 목록과 상세화면을 제공합니다.

이러한 가젯들을 조합하여 사용자 별로 원하는 화면을 꾸밀 수 있습니다.
