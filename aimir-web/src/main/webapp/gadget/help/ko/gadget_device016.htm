장비에서 내린 명령을 기준으로 검색 조건(분 단위)에 해당되는 명령 이력과, 통신 이력, 이벤트 이력 정보를 제공합니다.

 

미니 가젯

 

조회조건( 날짜, 검색 범위)에 따라 명령이 발생한 시점 이후부터 검색범위(분 단위) 이내에 발생한 명령 이력정보와 통신이력정보, 이벤트 이력정보를 제공합니다.

( * 통신 이력은 타입이 (C)Command 인 경우만 제공하고 이벤트 이력은 Alert/Event Class Name 종류가 Energy Level Changed 인 경우만 제공합니다.)

 

Log Analysis Mini Gadget

 

발생 시각

명령을 실행한 시간을 의미합니다.

타입

발생한 로그 타입을 의미합니다.

   ① O : 명령 이력(Operation)

   ② C : 통신 이력(Comunication)

   ③ E : 이벤트 이력(Event)

장비

이력이 발생한 장비를 나타냅니다.

명령

장비에 실행시킨 명령을 의미합니다.

결과

실행시킨 명령의 성공/ 실패 여부를 의미합니다.

메시지

이력 발생시 남겨진 메시지를 의미합니다.

 

 

맥스 가젯

 

로그분석 Max 가젯은 아래와같이 4가지 패널로 구성됩니다.

1. 조회 조건

2. 오퍼레이션 로그

3. 통신로그

4. 이벤트 알람로그

 

Log Analysis Max Gadget

 

1. 조회 조건

   날짜, 시, 분단위 검색조건 과 명령의 종류(오퍼레이션 코드), 장비 ID를 조회조건으로 사용 할 수 있습니다.

 

2. 오퍼레이션 로그 (명령 이력)

   조회 조건에 따라 발생한 명령 이력의 목록을 보여줍니다.

순번

명령이력의 순번을 의미합니다.

발생 시각

명령을 실행한 시간을 의미합니다.

장비

명령을 실행한 장비의 아이디를 의미합니다.

수행자

명령을 실행시킨 사용자의 아이디를 표시합니다.

명령

장비에 실행시킨 명령을 의미합니다.

결과

- 실행시킨 명령의 성공/ 실패 여부를 의미합니다.

- 실패했을 경우 에러 메세지가 표시됩니다.

 

3. 통신 로그 (통신 이력)

   1) 조회 조건에 따라 발생한 통신 이력의 목록을 보여줍니다.

   2) 명령 이력을 클릭시 해당 명령이 발생한 시각 이후로 검색범위(분 단위) 이내에 발생한 통신이력(동일한 장비, 동일한 명령)을 보여줍니다.

   3) 명령 이력은 타입이 (C)Command 인 경우만 조회 됩니다.

순번

통신 이력의 순번을 의미합니다.

발생시각

통신이력이 발생한 시간을 의미합니다.

송신

MCU ID, Sensor EUI64, FEP IP들의 주소

장비

받는쪽의 장비

명령

발생한 명령 코드

결과

성공/실패여부를 표시합니다.

 

4. 이벤트 알람 로그 (이벤트 발생 이력)

   1) 조회 조건에 따라 발생한 이벤트 발생 이력의 목록을 보여줍니다.

   2) 통신이력을 클릭시 해당 통신이 발생한 시각 이후로 검색범위(분 단위) 이내에 발생한 이벤트이력(동일한 장비)을 보여줍니다.

   3) 이벤트 이력 조회는 Alert/Event Class Name 종류가 Energy Level Changed 인 경우만 조회 됩니다.

순번

이벤트 이력의 순번을 의미합니다.

발생 시각

이벤트 이력이 발생한 시간을 의미합니다.

장비

이벤트가 발생한 장비의 아이디

메시지

이벤트에 대한 상태 표시.
