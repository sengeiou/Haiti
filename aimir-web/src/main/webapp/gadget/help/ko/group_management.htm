그룹 관리

 Top 

사용자 그룹과는 별도로 AiMiR 에서는 검침 장비, 장소 등을 종류별로 그룹을 만들어 관리 할 수 있습니다. 그룹을 만들 수 있는 그룹타입은 Location, Operator, Contract, MCU, Modem, Meter, EndDevice 가 있습니다. 그룹 관리 가젯에서는 원하는 타입의 그룹을 생성 후 그룹에 속한 멤버들을 추가/삭제 할 수 있습니다.

 

그룹관리 가젯은 미니 가젯과 맥스 가젯의 기능이 동일합니다.

 

그룹 관리 가젯의 첫 화면은 시스템에서 관리하고 있는 그룹의 목록을 테이블에 보여줍니다. 그룹에 속한 멤버가 있을 경우에는 그룹 이름 왼쪽에 플러스 아이콘이 생기고 오른쪽에는 멤버의 수를 표시합니다. 아이콘을 클릭하면 그룹 이름 아래로 멤버목록이 펼쳐집니다. 다시 를 클릭하면 멤버 목록이 사라집니다.

•그룹 추가 : 처음 가젯이 로드되었을 때 '그룹 추가' 버튼을 클릭합니다. 목록 마지막 줄에 새로운 항목이 생기고 빈 칸에 그룹의 이름 그리고 그룹 타입과 전체사용자 허용 유무를 선택한 후 '저장'을 클릭하면 새로운 그룹이 생성됩니다.

'취소' 버튼을 클릭하면 그룹생성 작업이 취소됩니다.

그룹 관리 가젯에서는 그룹별 멤버를 추가하거나 삭제할 수 있습니다. 멤버는 그룹 관리 창에서 별도로 입력하는 것이 아닌 시스템에 저장된 각종 데이터들이 멤버가 될 수 있습니다. 예를 들어 Location 타입의 그룹 A를 생성하였을 경우 그룹A에 포함시킬 수 있는 멤버들은 시스템에 저장된 모든 Location 들이 됩니다. Meter 타입의 그룹은 모든 Meter들이 멤버가 될 수 있습니다.

 

가젯 하단에는 멤버를 관리할 수 있는 화면이 있습니다. 아래 화면에서는 여러 타입의 멤버를 그룹에 추가하거나 삭제할 수 있습니다.

•멤버 검색 : 상단의 그룹목록에서 그룹 하나를 선택하였을 경우 하단 멤버관리 창에는 선택한 그룹의 그룹타입에 해당하는 멤버목록을 자동으로 보여줍니다. 특정 멤버를 찾고 싶으면 검색창에 검색할 멤버의 이름을 입력 후 검색버튼을 클릭합니다.
•멤버 추가 : 검색 결과에 해당되는 멤버들이 '검색 결과' 목록에 표시가 됩니다. 이 중 그룹에 추가할 멤버를 선택 후 오른쪽 화살표 를 클릭하면 '추가할 대상' 목록으로 이동하게 됩니다. 그리고 '저장' 버튼을 클릭하면 선택된 멤버들이 그룹에 추가됩니다.
•멤버 삭제 : 멤버를 그룹에서 삭제하려면 그룹 목록에서 멤버를 선택 후 '삭제' 버튼을 클릭하십시오.
