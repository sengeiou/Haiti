Time Diff.나 Compliance 검색 조건이 존재하는 경우

그래프는 그냥 해당하는 조건에 대해서 그린다.

Ex)

- Time Diff가 12~24로 선택된 경우

원형 그래프에는 Time Diff가 12~24인 전체 미터 개수를 표시

 

- Compliance가 Yes 인 경우

차트에는 Compliance가 Yes인 전체 개수를 표시

 

Method가 All인 경우는

Auto- 날짜 별 시간차의 백분율 그래프

Manual- 날짜 별 Status 상태 그래프를 보여줌
