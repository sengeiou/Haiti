이벤트 관리

 Top  Previous  Next

시스템 및 장비에서 발생한 모든 이벤트들에 대해 실시간 보기 및 이벤트 이력 조회, 이벤트 보기 설정 등에 대한 정보를 제공합니다.

 

미니가젯

 

이벤트관리 미니 버전은 2개의 탭로 구성되어 있습니다.

•장비 별
•이벤트 타입 별

 

장비 별 탭과 이벤트 타입 별 탭 모두 3개의 패널로 구성되어 있습니다.

1. 일별, 주별, 월별, 요일별, 계절별 조회 조건

2.  조회조건에 대한 차트

3. 조회조건에 대한 그리드
 

 

1. 일별, 주별, 월별, 요일별, 계절별 조회 조건

일별, 주별, 월별, 요일별, 계절별로 데이터를 조회 할 수 있습니다.

 

2.  조회조건에 대한 차트

조회조건에 대한 결과물이 장비 별 혹은 이벤트 타입별로 차트로 표현되어 집니다.

 

3. 조회조건에 대한 그리드

발생한 이벤트에 대해서 발생시각과 발생한 지역 그리고 메시지를 그리드로 볼 수 있도록 출력됩니다.

 

맥스가젯

 

이벤트관리 맥스 버전은 2개의 탭로 구성되어 있습니다.

•실시간 탭
•이력 탭

 

•[실시간 탭]

실시간으로 발생된 이벤트 내역을 보여줍니다.

메시지, 이벤트 발생 지역, 근원지 ID, 근원지 종류, 상태, 발생 시작, 복구시간, 지속시간을 위험 등급에 따라 색깔별로 표시됩니다.

 

•[이력 탭]

이벤트가 발생했던 이력을 보여줍니다.

장비/이벤트, 위헙 등급, 상태 등 다양한 조회 조건으로 검색이 가능합니다.

위험등급, Type, 메시지, 발생 지역, 근원지 ID, 근원지 종류, 상태, 저장 시각, 발생 시각, 복구시간, 지속 시간 등의 정보를 나타냅니다.

엑셀 출력 기능도 제공합니다.

 

Severity Level

Critical, Major, Minor, Warnin, Infomation, Normal 등 위험등급을 표시합니다.

Type

알람이 발생한 타입에 대해 Alert / Event 로 표시합니다.

Event Class Name

이벤트가 어떤 부분에서 발생했는지 표시합니다.

Class Name 종류는 하단을 참고.

Message

이벤트가 발생한 내용에 대한 메세지

지역

이벤트 발생지역

근원지 ID

이벤트가 발생한 장비의 아이디

근원지 종류

이벤트가 발생한 장비의 상세 타입

상태

이벤트에 대한 상태 표시.

- Acknowledged, Cleared, Cleared By Rule Automatically, Cleared Manually, Open, Open Manually

발생 시각

이벤트가 발생한 시간

복구 시간

이벤트가 종료(복구)된 시간

지속시간

이벤트가 발생했었던 기간, 이벤트 종료(복구) 시간 - 이벤트 발생 시간을 의미합니다.
