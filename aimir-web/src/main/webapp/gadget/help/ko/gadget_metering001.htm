검침차트

 Top  Previous  Next

 

전기, 가스, 수도, 열량 공타입 별 가젯으로 지역별 검침 률 현황과 각 미터의 검침데이터 수치를 검침기간별, 지역별, 고객별로 조회할 수 있는 기능을 제공합니다.

검침 상세에서 조회한 내용을 차트로 도표화하여 제공합니다.

 

미니가젯

 

검침차트 미니 버전은 아래와 같이 2개의 패널로 구성되어 있습니다.

1. 일별, 기간, 주별, 월별 ,계절별 데이터 조회

2. 검침 성공률 표시 차트 및 그리드

 

 

1. 일별, 기간, 주별, 월별 ,계절별 데이터 조회

일별, 기간, 주별, 월별, 계절별 데이터를 조회할 수 있습니다.

 

2. 검침 성공률 표시 차트 및 그리드

그리드 버전을 선택 시 왼쪽 화면이, 차트 버전을 선택 시 오른쪽 화면이 나타납니다.

•그리드 버전에서 검침성공률의 숫자는 검침 성공률(검침 성공횟수/검침 실패횟수)를 의미합니다.
•차트 버전에서 마우스를 막대에 가져다 대면 해당 막대그래프에 대한 정보가 표시됩니다.

검침차트 맥스 버전은 3가지 패널로 구성됩니다.

1. 시간별, 일별, 주별, 월별 , 요일별, 계절별, 연별 데이터 조회

2. 비교차트보기 기능, 엑셀 기능

3. 검침데이터에 대한 목록과 상세내역
 

 

1. 시간별, 일별, 주별, 월별 , 요일별, 계절별, 연별 데이터 조회

•시간별, 일별, 주별, 월별 , 요일별, 계절별, 연별 데이터를 조회할 수 있습니다.
•오른쪽 상단의 상세검색 버튼 클릭시 더 많은 조회 조건(지역, 장비 종류, 검침기 아이디, 집중기 아이디, 계약 종별)이 나타나 세분화된 조건으로 검색이 가능하게 됩니다.

 

2. [비교차트보기] 기능, [엑셀] 기능

•[비교차트보기] 기능

3번 패널에서 비교하고 싶은 계약을 2개이상 선택한후 [비교 차트 보기] 버튼을 누르면 두 계약을 비교한 상세정보가 화면에 출력됩니다.

[비교차트보기] 버튼을 클릭 시 창이 하나가 뜨게 되는데 이곳에 채널 별 사용량, 평균사용량, 최대 사용량, 탄소 배출량 등 상세 정보를 다양한 타입의 차트로 제공해줍니다.

 

채널아이디 : 설정한 채널 중 하나를 선택할 수 있습니다. 선택한 채널에 대한 정보가 차트 및 그리드로 표시됩니다.

선택 항목 : 사용량, 평균사용량, 최대사용량, 탄소 배출량[kg] 4가지 종류로 표현되며 여기서 선택한 정보에 대한 내용이 표시됩니다.

차트 보기 : 차트의 표현 방법을 선택할 수 있습니다.

3. 검침데이터에 대한 목록과 상세내역

- 이전[Kwh] 컬럼

현재 검색된 조회기간의 이전 기간의 사용량을 의미한다.

즉,  주별검색된 리스트일 경우 이전[kwh]는 한 주 전의 사용량을 의미하고,

일별검색된 리스트일 경우 이전[kwh]는 하루 전날의 사용량을 의미하며

시간별 검색된 리스트일 경우 02~04시까지의 시간을 검색했다면 이전[kwh]는 한 시간 전인 01~02 시까지의 사용량의 의미합니다.

- 검침기 아이디 : 검침된 장비의 ID, 예) 미터의 경우 미터아이디, 집중기의 경우 집중기 아이디를 의미

- 상세보기 컬럼

상세보기 컬럼의 상세보기 버튼을 클릭시 아래와 같은 창이 하나가 뜨게 되는데 이곳에 채널 별 사용량, 평균사용량, 최대 사용량, 탄소 배출량 등 상세 정보를 다양한 타입의 차트로 제공해줍니다.

 

상세보기는 크게 3개의 패널로 구성됩니다.

3-1.리포트 출력

3-2.계약의 기본정보

3-3.계약의 데이터에 대한 상세 정보

 

3-1.리포트 출력

•HTML, PDF, MS Word 로 리포트 출력이 가능합니다.
•HTML 리포트 출력 버튼을 누르게 되면 새로운 창이 뜨면서 그 전체에 상세보기와 같은 내용이 출력됩니다.

 

•PDF, MS Word 리포트 출력 버튼을 클릭하면 아래와 같이 상세보기와 같은 내용을 가진 레포트창이 뜹니다.
•버튼에 마우스를 가져다 대면 버튼에 대한 설명이 나타납니다.

 

버튼을 이용해 데이터를 파일로 사용자의 PC에 저장할 수 있습니다.

버튼을 누르면 아래와 같은 창이 뜨고 원하는 Format을 선택한 후 OK버튼을 눌러줍니다.

Export format은 Excel, PostScript, PDF, Word, PowerPoint 다섯 가지 타입을 지원합니다.

 

3-2.계약의 기본정보

계약에 대한 고객정보, 주소, 지역, 전화번호, 휴대폰번호, 미터타입, 미터아이디 등 기본 정보를 제공합니다.

※전기 같은 경우 Peak시간대를 구분하여 나타내며 나머지 가스 수도 열량의 경우는 하루 전체 사용량을 나타냅니다.

각 데이터는 일별데이터를 추출하며 각 Contract의 계약종별에 Peak Type에 따라 즉 계약종별로 Peak시간대가 달라짐. 여기서 Peak는 하루를 여러 시간대로 나눠서 해당 시간대의 전력 사용량을 나타냅니다.

예를 들면 Critical Peak (09~18시)까지의 사용량 , Peak (18~23시), Off Peak(23시~06시)까지 시간대별 사용량의 합을 나타냅니다.

즉 Critical Peak(최대부하)의 사용량, Peak의 사용량(중간 부하), Off Peak(경부하)의 사용량을 표시

만약에 Peak시간대가 존재하지 않는다면 하루 전체의 사용량을 표시
