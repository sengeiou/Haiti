맥스 가젯

 

맥스 가젯은 사용자 관리를 위해 더욱 상세한 정보를 관리할 수 있습니다.

이를 위해 그룹 정보, 그룹 사용자 정보, 로그인 이력으로 세분화한 정보를 제공합니다.

각 기능은 탭으로 구분이 되었으며 왼쪽 상단에 있는 리스트박스에서 그룹을 선택하면 해당 그룹의 정보를 확인하고 설정할 수 있습니다.

 

사용자 관리 맥스 버전은 3개의 탭으로 구성되어 있습니다.

•그룹정보 탭
•그룹 사용자 정보 탭
•로그인 이력 탭

 

•그룹 정보 탭

그룹 정보 탭은 3가지 패널로 구성되어 있습니다.

1. 사용자 그룹 선택

2. 기본 정보

3. 허용할 가젯 설정

 

AiMiR 3.0은 사용자 그룹을 생성하여 비슷한 종류의 사용자들을 하나의 그룹으로 묶어서 관리할 수 있습니다. 그룹 정보 탭에서는 그룹별로 접속 허용, 대시보드 권한, 검침데이터 및 시스템정보 읽기/쓰기 권한, Ondemand 명령의 권한 등을 설정 할 수 있습니다. 설정된 정보는 그룹에 속한 모든 사용자에게 적용이 됩니다. 그룹 정보에서는 가젯의 허용 유무 설정도 할 수 있습니다.  특정 그룹에 전체 가젯 중 전체 또는 일부만 허용을 할 수 있어 사용자의 종류별로 가젯의 접근을 제한할 수 있습니다

 

1. 사용자 그룹 선택

리스트 박스를 펼쳐 수정할 사용자 그룹을 선택합니다.

 

※ 새로운 사용자 그룹을 추가할 경우 오른쪽 상단의 [그룹 추가] 버튼을 눌러 추가합니다. 이때 [그룹 추가] 버튼은 [등록화면 닫기] 버튼으로 전환됩니다.

그룹 추가 화면으로 전환 후 처음 화면으로 돌아가고 싶다면 [등록화면 닫기] 버튼을 클릭하면 작업을 취소할 수 있습니다.

이때 [등록화면 닫기] 버튼은 [그룹 추가] 버튼으로 전환됩니다.

 

2. 기본 정보

•접속 허용 여부

로그인 권한을 주는 항목입니다.

이 항목을 No로 줄 경우 해당 사용자는 로그인 할 수 없는 기능으로 현재 적용되어 있지 않는 기능입니다.

 

•대쉬보드권한

대쉬보드 작성 권한을 줄건지 결정하는 항목입니다.

현재 적용되어 있지 않는 기능입니다.

 

•검침데이터

검침데이터에 대한 접근 권한을 설정합니다.

- 읽기 : 검침데이터를 확인만 할 수 있습니다.

- 쓰기 : 수정이 가능한 자료에 대해서는 데이터를 변경할 수 있는 권한을 제공합니다. 관리자에게만 추천되는 권한입니다.

- 명령 : Ondemand, UnitScanning 과 같은 원격 명령 실행에 대한 권한을 설정합니다.  '명령' 권한은 면 '읽기/쓰기' 권한도 포함합니다.

※ CDMA/GPRS 등의 통신 방식을 사용할 경우 원격 명령 수행 시 별도의 데이터 요금이 발생할 수 있으므로 주의가 필요합니다.

 

•시스템정보

- 읽기 : 시스템 정보를 확인할 수 있습니다

- 쓰기 : 시스템 정보를 수정할 수 있는 권한을 제공합니다.

- 명령 : 원격으로 시스템 정보를 확인할 수 있는 항목에 대해 원격 명령 권한을 제공합니다.

※ CDMA/GPRS 등의 통신 방식을 사용할 경우 원격 명령 수행 시 별도의 데이터 요금이 발생할 수 있으므로 주의가 필요합니다.

 

•설명

선택된 그룹에 대한 설명입니다.

 

•명령 실행

선택된 그룹에 허용하는 명령 리스트를 체크하는 부분입니다.

모뎀, 미터, 집중기 세 장비에 대한 명령의 종류를 체크해 선택된 그룹의 사용자가 사용할 수 있을 명령을 지정해줍니다.

 

3. 허용할 가젯 설정

새로 생성하는 사용자 그룹에 허용할 가젯을 선택할 수 있습니다.

오른쪽의 전체 가젯 목록에서 허용할 가젯을 선택한 후 왼쪽 화살표를 클릭하면 왼쪽의 허용된 가젯 목록에 추가됩니다.

취소하려면 허용된 가젯 목록에서 가젯 선택 후 오른쪽 화살표를 클릭하십시오

 

•그룹 사용자 정보 탭

그룹 사용자 정보 탭은 3개의 패널로 구성되어 있습니다.

1. 사용자 그룹 선택

2. 사용자 목록

3. 사용자 등록

 

1. 사용자 그룹 선택

리스트 박스를 펼쳐 등록/수정/삭제 하고자하는 사용자 그룹을 선택합니다

 

2. 사용자 목록

1번에서 선택한 그룹에 속한 사용자 목록을 볼 수 있습니다.

사용자 목록에서 사용자를 선택하면 3번 패널이 수정/삭제가 가능한 페이지로 전환 됩니다.

 

3. 사용자 등록

a) [사용자 등록] 버튼을 누르면 그룹에 추가할 사용자의 정보를 입력할 수 있는 화면이 3번패널에 나타나게 됩니다.

b) 전환되어 나타난 화면에 사용자를 추가할 때 필요한 정보들을 입력한 후 [저장] 버튼을 툴러 사용자를 등록합니다.

•아이디  :   추가할 사용자의 아이디를 입력합니다. 이 아이디는 로그인시 사용됩니다.

                   입력후에는 '중복확인' 버튼을 클릭하여 사용 가능 여부를 확인합니다.

•사용자명  : 사용자의 이름을 입력합니다.
•별명  :  사용자의 별명을 입력합니다.
•비밀번호/비밀번호 확인  :  비밀번호를 입력합니다. 로그인 시 사용됩니다.
•Email 주소  :  사용자의 Email 주소를 입력합니다.
•전화번호  :  사용자의 번화 번호를 입력합니다.
•접속 제한  :  'YES' 를 선택할 경우 해당 사용자의 로그인을 허용하지 않습니다.
•지역 : 지역을 설정하면 해당 지역의 미터, 모뎀등의 정보만 볼 수 있습니다.
•지역 제한 : 지역설정에 따라 정보를 제한할지를 선택합니다.
•접속 제한 사유  :  접속을 제한하는 사용자의 경우 이에 대한 설명을 입력할 수 있습니다.

 

(3) 로그인 이력

선택한 사용자 그룹에 대해 그 그룹에 속한 사용자들 중에서 AiMiR 에 접속했던 사용자들의 접속 내역을 확인할 수 있습니다.

아이디, IP주소 항목에 조건을 두어 검색할 수 있습니다.

로그인/ 로그아웃/ 로그인 실패 세가지 상태에 따라 조회가 가능합니다.

사용자 아이디

사용자가 로그인한 아이디를 표시합니다.

사용자명

사용자의 이름을 표시합니다.

사용자 그룹

사용자가 속한 그룹을 표시합니다. 화면 상단에 빨간 박스가 표시된 부분에서 확인 가능합니다.

IP 주소

사용자가 로그인한 IP의 주소를 표시합니다.

로그인 시간

사용자가 로그인한 시간을 표시합니다.

로그아웃 시간

사용자가 로그아웃한 시간을 표시합니다.

상태

로그인/ 로그아웃/ 로그인 실패 세가지 상태로 표현됩니다.
