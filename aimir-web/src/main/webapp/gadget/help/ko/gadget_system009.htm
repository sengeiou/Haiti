미니가젯

 

그룹 관리 미니 버전은 2가지 패널로 구성됩니다.

1. 그룹 타입, 그룹 명 검색

2. 그룹 목록

 

 

1. 그룹 타입, 그룹 명 검색

그룹 타입과 그룹 명으로 등록되어 있는 그룹들을 검색할 수 있습니다.

그룹 타입의 종류 : Contract, EndDevice, Location, MCU, Meter, Modem, Operator

 

2. 그룹 목록

그룹 명, 그룹 타입, 전체사용자 허용, 멤버 개수 등의 정보를 표시합니다.

•전체 사용자 허용 권한

- Y일 경우 모든 사용자가 사용할 수 있고 N일 경우 해당 그룹은 만든 사용자만 그 그룹을 사용할 수 있습니다.

- 그룹가젯이 사용되는 가젯 목록

 스케줄 관리, 미터 관리, Energy Balance Monitoring

•멤버 개수는 그룹에 속한 멤버의 수를 의미합니다.

 

맥스가젯

 

그룹 관리 맥스 버전은 4가지 패널로 구성됩니다.

1. 그룹 타입, 그룹 명 검색

2. 그룹 추가, 복사, 삭제

3. 그룹 목록

4. 멤버 추가 및 삭제

 

 

1. 그룹타입, 그룹명 검색

그룹타입과 그룹명으로 그룹을 조회할 수 있습니다.

 

2. 그룹추가, 복사, 삭제

•그룹 추가

새로운 그룹을 추가할 수 있습니다.

이 때 그룹 명은 중복될 수 없습니다. (※ 홈 그룹 관리와도 그룹명은 중복 될 수 없습니다.)

위의 그림에서 처럼 행이 하나가 생기면 이 곳에 만들고자 하는 그룹의 정보를 입력하면 특별한 액션 없이 바로 저장이 됩니다.

 

•복사

그룹 내의 멤버까지 모두 복사됩니다.

이 때 그룹명 중복이 되지 않으므로 변경해야 합니다.

•삭제

선택된 그룹을 삭제합니다.

멤버가 존재하는 그룹은 삭제할 수 없습니다.

 

3. 그룹목록

그룹명, 그룹타입, 전체사용자 허용, 멤버 개수를 리스트로 보여줍니다.

•전체 사용자 허용 권한

- Y일 경우 모든 사용자가 사용할 수 있고 N일 경우 해당 그룹은 만든 사용자만 그 그룹을 사용할 수 있습니다.

- 그룹가젯이 사용되는 가젯 목록

 스케줄 관리, 미터 관리, Energy Balance Monitoring

•멤버 개수는 그룹에 속한 멤버의 수를 의미합니다.

 

4. 멤버 추가 및 삭제

그룹을 선택하면 왼쪽 박스에 타입에 맞는 장비들이 검색되어 보여집니다.

그룹에 속한 멤버들을 오른쪽 박스에서 볼 수 있습니다.

•멤버추가

왼쪽 박스에서 추가하고자 하는 멤버를 선택 후 버튼을 눌러 멤버를 추가시킵니다.

여러 개의 멤버를 동시에 추가 할 수 있습니다.

•멤버삭제

오른쪽 박스에서 삭제하고자 하는  멤버를 선택 후 버튼을 눌러 멤버를 제외시킵니다.

여러 개의 멤버를 동시에 삭제 할 수 있습니다.
