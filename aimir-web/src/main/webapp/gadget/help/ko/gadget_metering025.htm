빌링리포트(전기)

 Top  Previous  Next

이 가젯은 미터에서 저장하는 TOU(Time Of Use)별 에너지 사용량 및 전체 에너지 사용량, 누적 에너지 사용정보를 엑셀로 출력할 수 있는 가젯입니다.

 

미니가젯

 

빌링리포트(전기) 미니 버전은 3가지 패널로 구성됩니다.

1. 보고서 타입 선택

2. 데이터 조건 설정

3. 엑셀 출력 기능

 

 

1. 보고서 타입 선택

보고서 타입은 일간 TOU, 월간 TOU, Current TOU 3가지 타입이 존재합니다.

일간 TOU : 일별로 TOU 데이터를 조회

월간 TOU : 매월 1일부터 말일까지의 TOU데이터를 조회

Current TOU : 실시간 TOU 데이터를 조회

 

2. 데이터 조건 설정

•1번 패널에서 보고서 타입을 선택하면 선택한 타입에 맞게 2번 패널의 구성이 변합니다.
•전일(전월)을 선택하면 데이터 포맷 형식이 주황색 으로 헤더부분이 표시되어 3번 패널에서 엑셀출력시 같이 출력됩니다.

    (Current TOU 타입의 경우에는 전일 비교(전월 비교) 기능을 지원하지 않습니다.)

•특정 지역의 고객에대해서만 조회하기를 원할 경우 Location의 텍스트 박스를 누르면 지역 Tree 형태의 컴포넌트를 보여주며

   특정 지역을 선택하면 특정 지역 및 하위에 속한 지역에 대한 데이터가 조회 됩니다.

   고객종별로 조회를 원할 경우 Contract 클래스의 Tariff Type 정보가 일치하는 고객을 조회합니다.

 

3. 엑셀 출력 기능

•2번 패널에서 조건 설정 후 검색버튼을 누르면 결과 필드에 데이터가 몇 건이 검색되었는지가 표시됩니다.

   검색버튼을 눌렀을때는 결과를 리스트로 보여주지 않고 전체 건수만 표현하며 실제 데이터 조회는 엑셀 버튼 클릭시 조회됩니다.

•데이터 검색 건수를 확인하고 건수를 제한하고 싶다면 출력범위 필드에서 건수를 제한 후 엑셀 버튼을 누르면 엑셀 파일을 다운받을 수 있습니다.

   예) 100건을 선택하면 100건 단위로 파일을 생성합니다.

 

 

맥스가젯

 

빌링리포트(전기) 맥스 버전은 미니버전과 동일한 기능을 가지며

미니 버전에서는 데이터 조회 후 결과를 리스트로 보여주지 않고 엑셀 출력 기능만 제공하는 반면 맥스버전에서는 조회 결과를 리스트로 보여줍니다.
 

 

조회조건대로 검색된 결과를 검침일, 고객명, 계약번호, 미터 아이디, 전체 에너지 사용량[kWh], 계약전력[kW], 수요전력[kW], Total kVah, Max Demand Time,

Max Demand[kVA], Import PhaseA[kWh], import PhaseB[kWh], Import PhaseC[kWh], 상세 보기 필드로 구분하여 화면에 나타냅니다.

 

이 때 상세보기 버튼을 클릭하면 아래와 같이 2개 혹은 3개의 패널로 이루어진 팝업 창이 생성됩니다.

 

1. 고객 정보

2. 전체 에너지 사용량 상세 정보

3. 전월 비교

 

 

1. 고객 정보

고객명, 계약 번호, 미터 아이디, 계약전력, 계약종별, 지역 정보를 표시합니다.

 

2. 전체 에너지 사용량 상세 정보

선택한 미터에 대한 전체 에너지 사용량에 대해 상세 사용량을 표시하여 보여줍니다.

 

3. 전일 비교(전월 비교 )

데이터 조회 시에  구성선택 필드에서 전일 비교(전월 비교) 체크박스를 체크했을 경우에만 나타나는 정보입니다.
