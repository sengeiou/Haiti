캘린더 차트

 

 

전기 같은 경우 Peak시간대를 구분하여 나타내며 나머지 가스 수도 열량의 경우는 하루 전체 사용량을 나타냅니다.

 

각 데이터는 일별데이터를 추출하며 각 Contract의 계약종별에 Peak Type에 따라

즉 계약종별로 Peak시간대가 틀려짐 여기서 Peak는 하루를 여러 시간대로 나눠서 해당 시간대의 전력 사용량을 나타냅니다.

 

예를 들면 Critical Peak (09~18시)까지의 사용량 , Peak (18~23시), Off Peak(23시~06시)까지 시간대별 사용량의 합을 나타냅니다.

 

즉 Critical Peak의 사용량, Peak의 사용량, Off Peak의 사용량을 표시

만약에 Peak시간대가 존재하지 않는다면 하루 전체의 사용량을 표시

 

 

 

 

비교 차트

 

-Tariff Chart

 

업무 시간대와 비 업무 시간대 등 하루 24시간을 특정 기준에 따라 여러 Tariff로 나누어 (현재는Business Hour, Non Business Hour) Tariff별로 사용량 정보 등을 표시합니다.

 

 

-Load Duration Chart

 

특정 사용량이 일정 기간내의 사용량 중에서 차지하는 백분율. 만약 Graph가 좌측에서 경사로 올라가는 경우 baseline 대비 Peak demand가 높고 빈도가 낮다는 것을 의미한다.

 

-Day Overlay Chart

 

일정 기간내의 일자에 대해 시간별 사용량 Graph를 한 화면에 Overlay시킴으로써 일자별 사용량 패턴 및 특이한 사용량 패턴을 보인 일자를 쉽게 식별할 수 있습니다.
