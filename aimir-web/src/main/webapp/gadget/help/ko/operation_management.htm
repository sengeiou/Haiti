Operation Management

 Top 

장비에 내린 명령(Operation)들의 목록과 각각의 시도 횟수, 성공률 등을 차트로 제공

장비 별로 시스템에서 제어할 수 있는 명령들에 대한 설정 및 시스템에서 지시한 명령 수행 이력들에 대한 정보를 제공

 

 

•Mini Gadget

 

•Max Gadget

 

 

Operation 목록에서 특정 Operation을 선택하면 그 Operation에 대해 검색 구간 동안의 Operator 별 실행 횟수(성공/실패)가 표시됨

 

아무것도 선택하지 않은 상태에서는 전체 Operation에 대한 성공 실패가 표시되도록 함

 

검색 기간에 따라 x축이 변하도록 함

하루 구간인 경우는 시간 축

일주가 넘어가는 경우는 주 별

한 달이 넘어가는 경우는 월별을 기본으로 그려주고 클릭 시 점점 디테일한 정보로 들어갈 수 있도록 구현

엑셀 출력 기능을 제공합니다.

 

 

Operator 탭을 클릭하면 Operation 목록이 트리 형태로 표현되고 해당 Operation을 클릭하면 우측 화면에 해당 오퍼레이션을 가지고 있는 장비들이 나타남

 

Equipment 탭을 클릭하면 장비 리스트가 표현되고 해당 장비를 클릭하면 해당 장비가 지니고 있는 오퍼레이션이 표시됨

 

공사별로 각각의 Operation Log Setting을 지닐 수 있고,

공사별 Operator가 Operation Log Setting의 변경 권한을 가진다

(변경 권한을 지니지 않은 사용자는 조회만 가능하고 변경은 불가능)

 

각 모델 별로 실행 가능한 Operation 목록을 트리 형태로 표시하고 해당 오퍼레이션 앞에 체크박스를 두어서 Logging 여부를 선택할 수 있게 함

 

유저 그룹 관리에서 enable한 Operation 들에 대해서만 로그를 Enable/Disable 할 수 있도록 한다
