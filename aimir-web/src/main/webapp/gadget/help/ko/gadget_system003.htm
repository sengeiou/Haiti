2. 계약 정보

맥스 가젯
 

공급사 정보들이 기본 정보, 공급사 관리 구역, 공급사 요금관리 탭으로 구분되어 보다 상세한 정보를 확인 할 수 있습니다.

 

공급사 관리 맥스 버전은 3개의 탭으로 이루어져 있습니다.

•기본정보 탭
•지역 탭
•Tariff 탭

 

•기본 정보

공급사 기본 정보 및 공급사가 제공하는 에너지 타입(전기, 가스, 수도, 열량 등) 설정 정보를 제공합니다.

- '공급사' 의 기본 정보

공급사명

공급사의 이름

설명

공급사에 대한 설명을 입력합니다.

전화 번호

공급사의 전화번호를 입력합니다.

국가

공급사가 위차한 지역을 선택합니다.

공급사 위치

공급사의 주소를 입력합니다.

사용언어

홈페이지에 적용할 언어를 선택합니다.

타임존

홈페이지에 적용할 Time Zone을 선택합니다.

MD Pattern

검침사용량 (kw, kWh, W, Wh, gcal, m3, mcal 등) 포맷에 적용되는 패턴입니다. 단, 온도, 압력, 퍼세트는 제외됩니다.

CD Pattern

통화(사용요금 등) 포맷에 적용되는 패턴입니다.

MD Round

검침사용량에 적용되는 소수점 처리 방식을 지정합니다. Down, Celling, Half Up 세 가지 방식을 지원합니다.

CD Round

통화에 적용되는 소수점 처리 방식을 지정합니다. Down, Celling, Half Up 세 가지 방식을 지원합니다.

 

- 공급 타입 : 오른쪽 화면구성에서는 각 에너지 타입별로 결제일과 Co2 배출량을 표시합니다.

•에너지 타입에는 Electricity, Water, Gas, Heat, VolumeCorrector 5가지 종류가 있습니다.

 

•빨간색 네모 박스가 쳐져 있는 부분에서 공급 에너지 타입과 탄소 배출량 이름, 과금일을 설정하고 버튼을 누르면 하단에 해당 에너지 타입과 과금일, Co2 정보가 표시됩니다.
•각 에너지 타입별로 하나의 과금일과 Co2배출량을 표시할 수 있습니다. 즉, Electricity에 관해 과금일과 Co2 배출량을 한번 설정하면 같은 타입의 에너지에 대해 새로운 정보를 추가할 수 없고 기존에 추가된 정보에 수정만 가능합니다.
•수정하고자 하는 타입정보에 마우스를 가져다 놓으면 버튼이 나타납니다.

왼쪽 버튼을 누르면 값을 수정할 수 있고 오른쪽 버튼을 누르면 값을 삭제할 수 있습니다.

 

•지역 탭

공급사 관리 구역을 보여줍니다.

공사가 관리하는 지역(행정상 )의 지역정보 설정 기능 제공 합니다.

 

지역 탭은 아래와 같이 2가지 패널로 구성되어 있습니다.
1. 지역 기본 정보
2. 지역 기본정보 서비스

 

 

1.지역 기본 정보

•공급사가 관리하는 구역의 기본 정보를 보여줍니다.
•지역정보가 트리형태로 구성되어 있습니다.
•트리형태를 펼쳐 원하는 지역을 선택하면 해당 지역에 대한 상위지역, 구역명, 순번 등의 정보가 나타납니다.

상위지역

선택한 지역의 상위 지역을 표시합니다.

구역명

선택한 지역의 이름을 표시합니다.

순번

- 선택한 지역이 현재 트리에서 몇번째 순서에 위치해있는지를 표시합니다.

- 하위노드가 새로시작되면 순번 또한 하위노드부터 리셋되어 표시됩니다.

 

2. 지역 기본정보 서비스

•1번에서 선택한 지역의 기본정보를 보여줍니다.
•Electricity, Heat, Gas 세 가지 타입에 대한 계약공급량을 설정/수정/삭제 및 확인할 수 있습니다.
•선택한 지역별 공급 에너지 타입과 계약 공급량을 설정하고 버튼을 누르면 하단에 해당 에너지 타입과 공급 사용량 정보가 하단에 표시됩니다.
•수정하고자 하는 타입정보에 마우스를 가져다 놓으면 버튼이 나타납니다.

왼쪽 버튼을 누르면 값을 수정할 수 있고 오른쪽 버튼을 누르면 값을 삭제할 수 있습니다.

 

•Tariff 탭

공급사의 요금을 관리하는 탭입니다.

에너지 종류, 적용 일자 별로 공급사의 계약 종별(Tariff) 요금 정보를 확인할 수 있습니다.
(현재는 해당 금액에 대해 수정할 수 있으며 계약 종별(Tariff)에 대한 추가 삭제 관리기능은 추후 서비스 필요 여부에 따라 추가 예정 입니다)

 

Tariff 탭은 아래처럼 3가지 패널로 구성되어 있습니다.

1. Tariff 검색

2. Import/Export 기능

3. Tariff 검색 결과

 

 

1. Tariff 검색

현재 전기, 가스,수도 에 대한 요금 정보를 제공하고 있으며 신규 공급사 추가 시 공급사별 요금 정보가 기초데이터로 갱신되어야 합니다.

공급사별로 요금이 최소 1년에 한번씩 갱신되므로 적용일자 별로 요금표를 제공합니다.

 

2. Add/Delete/Export 기능

•Add 버튼 : Tariff 정보를 추가할 수 있습니다.

 a. 오른쪽 상단의 Add 버튼을 클릭합니다.

 

 

 b. Add 버튼을 누르면 그리드의 마지막 줄에 행이 하나가 추가됩니다.

 

 c. 추가하고자 하는 정보들을 입력 후 오른쪽 하단에 있는 OK 버튼을 눌러 저장합니다.

 [Tariff] 컬럼에 입력한 값이 기존에 있는 Tariff 정보가 아닐경우 Tariff가 새로 생성되고,

  기존에 있는 Tariff 명을 입력했을 경우 그 정보를 가지고 나머지 입력한 값들만 추가로 등록됩니다.

 단, Season, TOU Rate, Supply Size 정보는 입력할 수 없습니다.

 

 d. 추가에 성공할 경우 아래와 같은 성공 메세지가 뜹니다.

 

 

•Delete 버튼 : Tariff 정보를 삭제할 수있습니다.

 

 a. 삭제하고자 하는 정보의 오른쪽 끝에 위치한 Del 컬럼의 체크박스에 체크합니다.

 

 

 b. 그리고 오른쪽 상단에 위치한 Delete 버튼을 클릭합니다.

 

 

 c. Yes 버튼을 누를경우 삭제가 진행되며 취소를 원하면 No 버튼을 누릅니다.

         

 

 d. 삭제에 성공할 경우 아래와 같은 성공 메세지가 뜹니다.

 

 

 ※ 삭제를 진행할 때 [Tariff] 컬럼에 해당하는 모든 정보를 삭제하고자 할 경우

 Season, TOURate, SupplySize, ServiceCharge, Admin Charge등 [Tariff] 컬럼의 정보를 제외한 나머지 정보들이 먼저 삭제되고,

 그 후에  [Tariff] 컬럼을 다시 삭제해 주어야 합니다.

 즉,  아래와 같이 Home Power(Low Voltage)를 삭제한다고 가정했을때 삭제를 실행하고 나면

 

 성공메세지를 뿌려주고 난 후 [Tariff] 컬럼에 대한 정보만 남게됩니다.                                                                                                                

 

 이 상태에서 Tariff의 Type에 대한 요금정보를 입력하려면 더블클릭을 통해 값을 수정하거나

 하나이상의 요금정보를 입력할 경우 Add 버튼을 이용해 요금정보를 추가한 후 OK 버튼을 누르면됩니다.

 최종적으로 Tariff의 Type도 마저 지우려면 다시 동일한 방법으로 삭제동작을 실행시켜주면 됩니다.

 

•Export 버튼

6번에 보여지는 데이터들을 엑셀 파일로 출력합니다.

 

•Update 하기

Update 동작의 경우 Add, Delete, Export 처럼 버튼을 통해 하는 것이 아니라 수정하고자 하는 데이터 필드를 더블 클릭한 후

해당 필드에 포커스가 가게되면 원하는 값을 입력한 후 화면 오른쪽 하단의 OK버튼을 누르면 됩니다.

   ※ 단, Tariff, Season, TOU Rate, Supply Size 항목에 대해서 수정할 수 없습니다.

 

 

3. Tariff 검색 결과

•검색결과를 화면에 표시합니다.
•적용일자가 선택되어 있지 않을 경우 가장 최근에 등록된 요금정보를 보여줍니다.
•요금 정보를 부분적으로 변경하고 싶을 땐 변경하고 싶은 데이터를 더블클릭해 데이터를 변경 후 오른쪽 하단의 [OK] 버튼을 눌러줍니다.
•Tariff, 계절, TOU Rate, 공급사이즈 항목에 대해서 수정할 수 없습니다.

Tariff

Tariff Type 명란을 표시합니다.

계절

계절별로 적용되는 Tariff가 다를경우 봄, 여름, 가을, 겨울 4가지 타입으로 나누어 표시됩니다.

모든 계절이 같은 요율로 적용될 경우에는 빈값이 들어갑니다.

TOU Rate

Critical Peak, Off Peak, Peak 등 Peak 타입을 표시합니다.

시간대에 따라 Peak 타입이 다르게 표시되며 이 Peak 타입에 따라 요율이 달라집니다.

공급사이즈

공급 사이즈를 표시하는 칸으로 예를 들어 50 Excess 라고 표기되면 공급량이 50이 초과될때 적용되는 요율을 의미합니다.

서비스 요금

매일 지불해야 하는 금액으로 한달에 한번 한꺼번에 몰아서 청구됩니다.

즉, 한달에 한번 서비스 요금(매일내야하는 금액) *한달 일수를 해서 고객에게 청구되어집니다.

관리 비용

서비스 요금처럼 매일 지불해야 하는 금액으로 한달에 한번 한꺼번에 몰아서 청구됩니다.

일반 고객(가정)에게는 적용되지 않는 개념으로 delivery point에 청구되는 비용입니다.

Trasmission Network Charge

매달 지불해야하는 값으로 delivery point의 annual utilised capacity에 근거한다.

Distribution Network Charge

배전망(전기를 분배하는 망)에 대한 사용 요금.

에너지 수요 요금

계약전력(kW) 호은 공급사이즈에 따른 기본 요금을 의미합니다.

Active Energy Charge

유효 사용량(kWh)에 따른 요금을 의미합니다.

Reactive Energy Charge

무효 샤용량(kWh)에 따른 요금의 의미합니다.

Rate Revalanceing Levy

신재생 에너지를 제외한 일반 원자력, 화력 등으로 생산되는 에너지에 대해 부과되는 세금
