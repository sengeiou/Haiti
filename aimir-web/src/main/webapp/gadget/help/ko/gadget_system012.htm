홈 그룹 추가/수정/삭제, 그룹멤버 관리 가젯입니다.

 

미니가젯

 

홈 그룹 관리 미니 버전은 2가지 패널로 구성됩니다.

1. 그룹 타입, 그룹 명 검색

2. 그룹 목록

 

 

1. 그룹 타입, 그룹 명 검색

그룹 타입과 그룹 명으로 등록되어 있는 그룹들을 검색할 수 있습니다.

 

2. 그룹목록

그룹 명, 그룹 타입, 집중기 아이디, 멤버 수 등의 정보를 표시합니다.

멤버 개수는 그룹에 속한 멤버의 수를 의미합니다.

 

맥스가젯

 

홈 그룹 관리 맥스 버전은 5가지 패널로 구성됩니다.

1. 그룹 타입, 그룹 명 검색

2. 그룹 추가 및 삭제

3. 그룹 목록

4. 그룹에 추가할 수 있는 멤버 목록

5. 그룹에 추가된 멤버 목록

 

 

1. 그룹 타입, 그룹 명 검색

그룹 타입과 그룹 명으로 등록되어 있는 그룹들을 검색할 수 있습니다.

홈 그룹가젯의 그룹 타입의 종류 : HomeGroup, IHD

 

2. 그룹 추가 및 삭제

•그룹 추가

새로운 그룹을 추가할 수 있습니다.

이 때 그룹 명은 중복될 수 없습니다. (※ 그룹 관리와도 그룹명은 중복 될 수 없습니다.)

•삭제

선택된 그룹을 삭제합니다.

멤버가 존재하는 그룹은 삭제할 수 없습니다.

 

3. 그룹 목록

그룹명, 그룹타입, 집중기 아이디, 멤버 개수를 리스트로 보여줍니다.

•집중기 아이디

해당 그룹을 등록할 집중기의 아이디를 의미합니다.

 

4. 그룹에 추가할 수 있는 멤버 목록

선택된 그룹에 추가할 수 있는 멤버 목록들을 보여줍니다.

여러개의 멤버를 선택할 수 있으며 버튼을 통해서 그룹에 멤버를 추가합니다.

멤버를 처음 추가할 때에는 집중기에 해당 멤버가 등록되지 않으며 5번 패널에서 [등록] 버튼을 통해 집중기에 등록해야 합니다.

 

5. 그룹에 추가된 멤버 목록

이미 그룹에 추가된 멤버들의 목록을 보여줍니다.

•[집중기 등록 결과] 컬럼

현재 집중기에 해당 멤버가 등록되어 있는지를 나타냅니다.

•[최종 동기화 시각] 컬럼

마지막으로 집중기와 동기화한 시각을 표시합니다.

 

여러 개의 멤버를 선택할 수 있으며 버튼을 통해서 그룹에서 멤버를 삭제할 수 있습니다.

멤버 삭제시 해당 멤버의 DCU 등록 결과가 [등록] 상태이면 삭제할 수 없습니다.

 

•[등록] 버튼

멤버를 선택 후 등록 버튼을 누르면 집중기에 해당 멤버가 등록된 후 [집중기 등록 결과] 컬럼이 [해지] 에서 [등록]으로 바뀝니다.

•[해지] 버튼

멤버를 선택 후 해지 버튼을 누르면 집중기에 해당 멤버가 해지된 후 [집중기 등록 결과] 컬럼이 [등록] 에서 [해지]로 바뀝니다.

•[집중기 멤버 동기화] 버튼

멤버를 선택 후 집중기 멤버 동기화 버튼을 누르면 해당 정보를 집중기에서 검색 후 현재 집중기에 해당 멤버가 등록되어 있는지 여부를 [집중기 등록 결과] 컬럼에 표시합니다.
