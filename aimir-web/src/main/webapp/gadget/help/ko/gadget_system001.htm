시스템에서 관리하는 코드의 정의 및 코드 목록, 코드 정보 추가/수정 삭제 기능을 제공합니다.

 

미니 가젯

 

코드관리 미니 버전은 코드 내역을 볼 수 있고 코드에 대한 검색 기능을 제공합니다.

 

 

맥스 가젯

 

시스템에 등록된 코드 정보를 간단히 검색 및 추가 / 삭제 하는 기능을 제공합니다.

 

코드관리 맥스 버전은 크게 2개의 패널로 구분됩니다.

1. 검색 기능

2. 코드내역리스트 및 추가/수정/삭제 기능

 

 

1. 검색 기능

검색창에 검색하려는 코드 명을 입력 후 검색버튼을 클릭하면 해당되는 항목은 빨간색으로 표시됩니다.

하위 내역까지 검색됩니다.

 

2. 코드내역리스트 및 추가/수정/삭제 기능

코드의 정보를 보여주며 setting 항목에서 코드의 Add/Update/Delete가 가능합니다.

 

•Add

a) 추가하고자 하는 코드의 상위 코드를 찾아 setting필드의 Add 버튼을 눌러줍니다.

   ※ 만약 추가하고자 하는 코드가 이미 존재하고 있는 코드의 하위코드가 아닌 가장 상위의 코드라면 RootAdd 버튼을 클릭합니다.

 

b) Add 버튼을 누르게 되면 아래 그림처럼 코드 정보를 입력할 수 있는 레코드가 생기는데 이 곳에 추가하고자 하는 코드 정보를 입력하고 save 버튼을 누릅니다.

 

•Update

a) 수정하고자 하는 코드 레코드에 setting 필드에서 Update버튼을 누릅니다.

 

b) 버튼을 누르면 아래 그림처럼 코드정보가 수정가능하도록 화면이 바뀝니다.

   원하는 내용으로 수정한 후 Save 버튼을 클릭합니다.

 

•Delete

a) 삭제하고자 하는 코드의 setting 필드에서 Delete 버튼을 누릅니다.

이 버튼을 클릭하면 경고창이 나타나 삭제할 것인지 다시한번 묻습니다. 이때 Yes를 클릭하면 코드가 삭제됩니다.
