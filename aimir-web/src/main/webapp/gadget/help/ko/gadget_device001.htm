집중기 관리

 Top  Previous  Next

공사에서 관리하는 집중기 목록 및 상세 정보,집중기 별 검침스케줄 설정,집중기 설치 위치 정보,집중기 설정 및 제어 등의 기능을 제공

 

미니 가젯

 

집중기관리 미니 버전은 아래와 같이 2개의 패널로 구성되어 있습니다.

1. 집중기의 타입 및 통신상태별 차트

2. 집중기의 타입 및 통신상태별 그리드

 

 

집중기 타입별(Indoor, Outdoor, DCU)/통신 상태별(24시간 이내 통신 정상, 24시간 이상 통신 장애,  48시간 이상 통신 장애 등)으로 차트와 그리드를 제공합니다.

 

맥스 가젯
 

집중기관리 맥스 버전은 아래와 같이 3개의 패널로 구성되어 있습니다.

1. 집중기 타입, SW/HW 버전, 설치일시, 통신타입, 지역 등 조건 입력

2. 조회조건에 따라 검색된 집중기 목록

3. 선택한 집중기에 대한 상세 정보

 

 

1. 집중기 타입, SW/HW 버전, 설치일시, 통신차입, 지역 등 조건 입력

집중기 타입, 집중기 아이디, SW/HW 버전, 설치일시, 통신타입, 지역 등 입력한 조건에 따라 해당하는 집중기를 검색합니다.

재입력 버튼 클릭 시 입력 중이던 조회조건들이 초기화 됩니다.

 

2. 조회조건에 따라 검색된 집중기 목록

•조회조건에 만족하는 집중기 목록을 보여주며 집중기 타입, 아이디, 이름, 제조사, 모델, 지역,  IP 주소, SW/HW 버전, 설치일시, 최종통신일시, 통신타입, 통신상태 등의 정보를 표시합니다.
•통신 상태별,최종통신 올림/내림 차순, 설치일시 올림/내림 차순 별로 필터링이 가능합니다.
•추가/ 삭제가 가능하고 조회된 집중기 목록을 엑셀로 출력 가능합니다.
•추가

a) 2번 패널 오른쪽 상단에 위치한 Add 버튼을 클릭하면 3번 패널의 위치에 새로 추가할 집중기에 대한 정보를 입력할 수 있는 화면이 나타납니다.

b) 새로 추가할 집중기에 대한 정보를 입력한 후 [확인] 버튼을 클릭하면 집중기가 추가됩니다.

- 빨간 별표(*)로 표시되어져 있는 항목은 필수 값으로 반드시 입력해야 하는 항목입니다.

- 집중기 아이디를 입력한 후에는 반드시 [중복확인] 버튼을 눌러서 중복값을 체크해 주어야 합니다. 집중기 아이디가 중복값일 경우에는 집중기가 등록되지 않습니다.

- 집중기의 이미지를 등록하고자 하는 경우에는 [검색]버튼을 눌러서 등록하고자 하는 이미지를 불러옵니다.

•삭제

a) 삭제하고자 하는 집중기를 클릭한 후 Delete 버튼을 클릭합니다.

 

3. 선택한 집중기에 대한 상세 정보

선택한 집중기의 기본정보와 선택한 미터의 위치 정보, 스케줄, 이력, 집중기 CODI 정보를 표시합니다.

상세정보는 4개의 탭으로 구성되어 있습니다.

•기본정보
•위치정보
•스케줄
•이력
•집중기 CODI

 

•기본정보

 

기본정보 탭은 아래와 같이 3개의 패널로 구성되어 있습니다.

  3-1. 기본 정보

  3-2. 설치 정보

  3-3. 장비점검

  3-4. 장비설정

 

 

  3-1. 기본 정보

 선택된 집중기의 아이디, 타입, 제조사, 모델 명을 표시하며 수정이 가능합니다.

 

  3-2. 설치 정보

  설치일시, SW/HW 버전, 최종 버전업 일시, 통신타입, IP 주소, 집중기 모바일, 지역 정보를 표시하며 수정이 가능합니다.

 

  3-3. 장비점검

  집중기 상세 정보에서는 Command 프로토콜을 이용하여 실시간으로 상태 정보를 확인 및 집중기에 대한 제어 기능을 제공합니다.

DCU 상태감시

Power State, Low Battery여부, Flash Total Size, Flash Use Size, Memory Total Size, Memory Use SIze, Current Temp, System Time, Sink Neighbor Node, Sink State 정보를 가져와 표시합니다.

 유닛 스캐닝

집중기의 설정의 세부 사항을 확인, System(DCU) Name, System Type, System Phone Number, System up Time, System Ethernet Type, System Mobile Type, System Mobile Mode, System HW Verstion, System SW Verstion 정보를 가져와 표시합니다.

집중기 자가진단

MCU State, Sink State, Power State, Temperature State, Memory State, Flash State, GSM State, Ethernet Ste 정보를 가져와 표시합니다.

  3-4. 장비설정

집중기 시간 설정 버튼을 누르면 집중기의 시간 설정이 가능합니다.

 

•위치정보

 

위치정보 탭에서는 설치된 집중기의 Map 정보를 Google Map 을 이용하여 표현합니다.

또한 집중기에 연결된 장비 리스트에 대해서도 확인 할수 있습니다.

•이력

이력 탭에서는 해당 집중기의 통신 로그, 장애 이력, 명령어 수행 이력등을 기간별로 조회할 수 있으며 엑셀로 출력이 가능합니다.

•각 History 별 항목 설명

- Comm. Log : Communication Log 가젯 내용을 간단히 표시

서비스 타입

송/수신된 서비스의 타입.

서비스 타입의 종류

- (A)Alarm

- (C)Command

- (D)Data File

- (E)Event

- (F)File Transport

- (M)Metering

- (N)Metering

- (P)Partial

- (R)Metering

- (S)Metering

- (T)Telnet

Out Bytes

송신량, byte 단위로 나타냅니다

In Bytes

수신량, byte 단위로 나타냅니다.

전체 바이트 수

Out Bytes + In Bytes 로 송/수신하는데 사용한 총 Byte를 표시합니다.

총 통신 시간

송/수신시 통신하는데 걸린 시간을 표시합니다.

 

- Alert History : Event Alarm Log 가젯 내용을 간단히 표시

중요도

Critical, Major, Minor, Warnin, Infomation, Normal 등 위험등급을 표시합니다.

메세지

이벤트가 발생한 내용에 대한 메세지

지역

이벤트 발생지역

발생시각

이벤트가 발생한 시간

복구시간

이벤트가 종료(복구)된 시간

지속시간

이벤트가 발생했었던 기간, 이벤트 종료(복구) 시간 - 이벤트 발생 시간을 의미합니다.

 

- Operation History : Operation Log 가젯의 내용을 간단히 표
