통신 장비 별 통신 타입, 통신 시간, 통신 양 등에 대한 정보를 제공합니다.

미니가젯
 

기본 조회에서는 일주일치의 통신 로그를 송수신으로 구분하여 챠트로 표현합니다.

 

맥스가젯

 

통신이력관리 맥스 버전은 4개의필드로 구성되어 있습니다.

1. 시간별, 기간, 주별, 월별 및 데이터 조회 조건

2. 데이터 타입에 따른 차트

3. 데이터 송수신량 차트

4. 조회 조건에 따른 통신이력 상세 목록

 

 

1. 시간별, 기간, 주별, 월별 및 데이터 조회 조건

시간별, 기간, 주별, 월별 및 세부 내용을 적요하여 해당 통신로그를 조회 할수 있습니다.

 

2. 데이터 타입에 따른 차트

•데이터 타입

(A)Alarm

(C)Command

(D)Data File

(E)Event

(F)File Transport

(M)Metering

(N)Metering

(P)Partial

(R)Metering

(S)Metering

(T)Telnet

 

3. 데이터 송수신량 차트

조회 조건에 따라 검색된 결과들의 송신량, 수신량의 합을 나타낸 차트입니다.

x 축 : 송/수신량을 표시하며 단위는 byte 로 나타냅니다.

y 축 : 송신량과 수신량 2가지를 나타냅니다.

 

4. 조회 조건에 따른 통신이력 상세 목록

로그시각, 데이터 타입, 프로토콜, Sender, Receiver, 송/수신량 결과, 총 통신 시간, 오퍼레이션 코드 등 상세한 목록을 제공합니다.

로그시각

송/수신량 시간

데이터 타입

송/수신된 데이터의 타입으로 종류는 2번 패널설명의 데이터 타입 참고

프로토콜

송/수신된 프로토콜 타입

Sender

MCU ID, Sensor EUI64, FEP IP들의 주소

Receiver

받는쪽의 IP 주소

송신량 [byte]

송신량, byte 단위로 나타냅니다.

수신량 [byte]

수신량, byte 단위로 나타냅니다.

결과

성공/실패여부를 표시합니다.

총 통신  시간

송/수신시 통신하는데 걸린 시간을 표시합니다.

오퍼레이션 코드

서비스 필드값으로 서버에서 내려가는 명령어의 경우 Cmdxxx 타입의 값이 들어갑니다.
