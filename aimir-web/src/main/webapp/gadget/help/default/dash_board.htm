•대시보드
해당 시스템은 대시보드에 여러 개의 가젯을 올려놓아 한 대시보드 안에서
가젯의 위치를 이동 시킬 수 있습니다.
사용자가 필요한 가젯만 보고 싶으면 보지 않는 가젯은 비 활성 시킬 수 있습니다.
사용자별 대시보드를 제공함으로써 사용자가 가젯의 위치변경 후 다시 로그인 시 해당 대시보드의 레이아웃이 그대로 유지됩니다.
 
각 가젯은 독립적으로 구동 되며 같은 이름의 가젯을 한 대시 보드에 여러 개 띄울 수 있습니다.
대시보드는 최대 8개까지 추가가능하나 브라우저의 특성상 4-5개까지 추천하며
하나의 대시보드당 가젯은 최대 16개까지 추가할 수 있으나 가젯의 로딩 속도가 느려지므로
그 이하로 사용할 것을 권장합니다.
 

대시보드에서 특정 가젯을 최대화 하면 대시 보드안에 해당 가젯이 최대화 되며 다시 해당 가젯을 최소화 하면 원래 화면 배치로 돌아오게 됩니다.
