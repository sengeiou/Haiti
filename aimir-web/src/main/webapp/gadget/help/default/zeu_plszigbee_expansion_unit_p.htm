ZEU-PLS(ZigBee Expansion Unit-Pulse)

 Top  Previous  Next

수도 계량기 같은 펄스형 계기와 기 정의된 Interface(Pulse Cable)를 통해 통신 경로를 설정하고 상위 시스템인 집중기와는 ZigBee를 이용하여 통신한다.

 

 

 

전기계기용 단말기의 경우와 동일하게 검침 데이터를 Event-Driven 방식으로 전송한다.

 

 

 

이 형태의 단말기는보통 배터리 모드로 동작하며 NAST-PDA를 통해 단말기 설치 및 설정 변경을 위새서는 외부에서 강제로 Wakeup시켜야 한다. 집중기를 통한 설정 조회 및 변경의 경우는 단말기가 깨어 동작하는 시간 즉 Active Time 내에서 수행된다.
