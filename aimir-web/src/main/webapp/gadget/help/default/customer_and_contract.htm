공사별 고객 정보, 계약 정보, 선불 고객 관리, 계약 변경 내역, 월별 사용량 등의 정보를 제공

 

•Mini Gadget

 

전체 고객 중 고객의 계약 상태에 따라 분류합니다.

 

Normal : 정상 계약 유지 고객

Pause : 일지 정지 고객

Stop : 사용중지 고객 (요금미납 차단 등등)

Cancel : 해지 고객

 

또한 전체 고객을 계약 종별(Tariff Type)로도 분류하여 보여줍니다.

 

•Max Gadget

 

미니 가젯에서 전체 최대화 하면 맥스 화면으로 전환되며 맥스 가젯 에서는 전체 고객, 전기,가스, 수도, 열량 등의 공 서비스 별 계약 고객 목록을 보여줍니다. 위 목록에서 특정 고객을 선택할 경우 하단의 상세 정보에 아래와 같이 계약 관련 상세정보 계약변경 이력, 선불 고객인 경우 선불 충전 내역, 각 계약 별 에너지 사용량 정보를 보여줍니다.

 

엑셀 출력 기능을 제공합니다.

 

•계약에 관련된 상세정보

•계약 변경에 관한 이력

 
•선불 고객인 경우 충전 관련 이력

 
•계약별 에너지 사용량

 

 

 

고객과 공사간의 계약관계를 표현하지만 미터 별로 다를 수 있기 때문에 미터와의 관계로 표현합니다.

계약 정보에는

- 계약종류 (공사 코드 참고)

- 결제종류 (후불, 선불)

선불 시 잔액

- Contract Demand사용 여부

- 신용 모드 (credit, dept)

 

Prepayment 고객 분류

사용량과 tariff 정보를 이용하여 사용금액을 계산하여 저장합니다. 잔액이 마이너스(debt) 상태가 되면 경고를 보내고 공을 차단합니다.

공 차단 오퍼레이션이 안될 수도 있기 때문에 재시도할 수 있도록 합니다.

 

Contract Number(계약 번호): 공사에서 부여하는 별도 계약 번호
