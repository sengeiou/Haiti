공급사에서 관리하는 지역관리, 공급사 정보, 공급사가 제공하는 공급 서비스(전기, 가스, 수도 등)에 대한 정보, 과금 정보 관리 등의 서비스를 제공.

 

•Mini Gadget
 

로그인한 사용자의 공사 정보를 기본으로 하며 로그인한 사용자가 특정 공사에 속해있지 않을 경우 공사 목록을 보여주며 특정 공사에 속해 있을 경우에는 해당 내용을 보여주지 않는다.
 

•Max Gadget
•Basic Information

공급사 기본 정보 및 공사가 제공하는 에너지 타입(전기, 가스, 수도, 열량 등) 설정 정보
 
•Supply Mgmt Area

공사가 관리하는 지역(행정상 )의 지역정보 설정 기능
 
•Supply Mgmt Pay

공사의 계약 종별(Tariff) 요금 (현재는 해당 금액에 대해 수정할 수 있으며 계약 종별(Tariff)에 대한 추가 삭제 관리기능은 추후 서비스 필요 여부에 따라 추가 예정이다.
 
현재 전기, 가스,수도 에 대한 요금 정보를 제공하고 있으며 신규 공사 추가 시 공사별 요금 정보가 기초데이터로 갱신되어야 합니다. 공사별로 요금이 최소 1년에 한번씩 갱신되므로 적용일자(Apply Date) 별로 요금표를 제공합니다. 엑셀 출력 기능을 제공합니다.
