VEE (Metering Data Validation, Estimation, Editing)

검침데이터에 대한 Validation Check, Editing, Estimation을 할 수 있는 기능을 제공하며 Editing, Estimation을 했던 이력 등을 조회할 수 있는 기능을 제공

 

•Mini Gadget

 
•Max Gadget

Editing, Estimation 항목은 Operator의 Role에 따라 수정권한이 있습니다. 즉 로그인한 사용자(Operator)의 Role 값을 읽어서 Role 속성 중 Authority가 (RW) 인 경우에 변경을 허용하게 합니다.

 

Editing이나 Estimation을 했다면 상태 플래그에 Edited 또는 Estimated 라고 표시가 되어야 하며 Rollback을 할 수 있게 버튼 혹은 링크가 있어야 합니다. Rollback을 하면 원래 대로 돌아간다. 단 Estimation의 경우에는 Rollback을 제공하지 않는다.

 

복사 이동을 하기 위해서는 먼저 체크박스에서 복사(이동대상을)선택하여 복사/이동 버튼을 클릭하게 되면 선택한 필드들이 선택할 수 없게 복사대상 혹은 이동대상으로 표시됩니다. 이후 복사 대상을 카피를 위해 카피할 대상을 체크박스로 선택 한 뒤 붙여 넣기를 클릭하면 이전에 복사한 대상이 타겟으로 복사가 됩니다. 만약에 한 행을 선택하였으나 복사 대상이 여러 개라면 일괄적으로 복사가 됩니다. 만약에 복사대상이 타겟 보다 더 많다면 예외메시지를 뿌리고 다시 선택할 수 있게 해야 합니다.

 

Editing버튼을 클릭하면 각 검침데이터의 Channel 필드를 수정할 수 있게 합니다.
