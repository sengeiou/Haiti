원격 검침 서버(NACS-Nuri AiMiR Center System)

 Top  Previous  Next

 

 

WEB User Interface를 통해 수행되는 사용자 명령을 수신하여 특정 장치(집중기, 단말기 또는 계기)로 전송하고 해당 결과를 다시  사용자에게 전달한다. 또한 주기적으로 수신되는 검침 데이터 및 관련 Event정보를 Database에 저장하고 통계 및 이력 등의 정보를생성한다.

 

NAST-PDA와 서버는 WLAN을 통하여 필요한 정보를 연계한다. 즉 단말기 설치 또는 운영시 기초 정보를 다운로드하거나 설치/설정 완료 후 해당 정보를 복잡한 절차없이 WLAN을 통해 서버 Database에 반영할 수 있다.
