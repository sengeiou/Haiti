Main window는 다음과 같은 요소로 구성된다.

 

•Top menu(①) : Top menu 탭을 클릭시 해당 화면으로 이동하며 각 화면에는 설정된 미니 가젯들이 나타난다.

 

•Toolbar image(②) : 로그인한 사용자의 아이디를 표시하며, Account를 클릭 시 해당 사용자의 상세정보 화면이 출력된다.
마지막 Logout을 클릭하면 Logout 된다.

 

•Gadget Setting Button(③) : 클릭시 가젯 설정 창으로 이동한다. 대시보드 설정과 각 가젯을 검색 및 설정할 수 있다.

 

•Mini Gadget(④) : 미니가젯. 각각의 개별적인 가젯들로 맥스가젯으로 전환이 가능하다. 미니가젯에서는 간략한 정보만을 나타낸다.
가젯 상단의 Toolbar image는 맥스가젯과 동일하며 마우스 클릭 시 차례대로, 가젯접힘, 도움말, 새로고침, 가젯전환, 가젯닫기 기능이 제공된다.

 

 

화면은 1280*1024 해상도(resolution)에 최적화되어 있다.
