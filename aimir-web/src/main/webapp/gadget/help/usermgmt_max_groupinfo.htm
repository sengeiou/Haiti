그룹 정보을 통해서 시스템에 등록된 모든 사용자 그룹 정보를 조회할 수 있다.

'그룹정보' 탭에서는 다음과 같은 기능을 갖는다.

1)새 그룹 등록
2)그룹별 기본 정보 조회 및 수정
3)그룹별 허용 가젯 설정 및 변경

 

 

[Figure 2-1] 사용자 관리 맥스 가젯 - 그룹정보

 

 

 

화면 항목의 설명은 다음과 같다.

▪그룹 명: Group 명.
▪접속 허용 여부: Y/N값을 가지며, N의 경우 이 User Group의 사용자는 Login을 허용하지 않음을 의미
▪읽기/쓰기/명령 권한: "추가/업데이트/삭제" 명령은 "쓰기"권한에 속하며 그 이외의 명령은 "읽기"명령에 속한다. 또한 "쓰기" 권한은 묵시적으로 "읽기" 권한을 포함한다. 권한은 단 한개만 선택이 가능하다.
▪허용된 가젯: 이 사용자 그룹에 허용된 가젯으로써, 사용자가 사용할 수 있는 가젯을 제한하게 된다.
▪전체 가젯 : 이 사용자 그룹에 허용될 수 있는 모든 가젯들을 출력한다.
▪설명: 사용자 그룹에 대한 설명

 

그룹 기본정보 수정

기본정보의 내용을 수정 후 하단의 "변경" 버튼을 클릭하면 해당 그룹의 기본정보가 수정된다.

 

새 그룹 등록

그룹의 기본정보 우측상단의 "새 그룹 등록"을 클릭하게 되면 [Figure 2-2]와 같이 새 그룹 추가 화면으로 이동하게 되며 각 기본정보가 입력 가능한 상태로 변경된다. 기본정보를 입력 후 하단의 "추가" 버튼을 클릭하면 새 그룹이 추가되며, "취소" 버튼을 클릭 시 이전 화면으로 돌아가게 된다.
