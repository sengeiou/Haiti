아래 그림처럼 WEB Browser의 주소 입력창에 WEB URL(예. https://187.1.10.200:8085/aimir-web/)을 입력하면 위와 같이 Login 화면이 생성된다. Login ID 및 Password 입력 부분에 각자에게 부여된 ID와 Password를 정확하게 입력하게 되면 Main화면이 표시된다.

 

[Figure 2] AiMiR Metering 로그인

 

 

만약 입력된 ID와 Password가 유효하지 않는다면, 오류 메세지가 출력된다.

"Remember me" 옆 체크박스를 클릭하면, 사용자가 입력한 ID 정보가 이후 로그인시에 동일하게 나타난다.
