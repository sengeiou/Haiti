rocksaw 설치 https://www.savarese.com/software/rocksaw/

1. lib install
   mvn install:install-file -Dfile=lib/rocksaw-1.1.0.jar -DgroupId=com.savarese -DartifactId=rocksaw -Dversion=1.1.0 -Dpackaging=jar

2. rocksaw so 생성
   cd src/main/cpp/rocksaw
   make
